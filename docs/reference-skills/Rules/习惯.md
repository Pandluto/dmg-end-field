# 输出习惯

## 格式要求

- 输出结果尽量用**控制台格式**，不用表格的 Markdown 格式
- 计算过程尽量详细，展示每一步的计算逻辑

## 默认值规则

- 五星角色默认满潜（潜能5）
- 六星角色默认0潜
- 角色默认90级
- 武器默认90级
- 珠子默认663（如未提及）

## 潜能效果区分

角色潜能效果需要区分无条件生效和条件生效：

**无条件生效**：
- 直接加到角色面板
- 有 `stats` 字段的潜能
- 例如：敏捷+15、物理伤害+8%

**条件生效**：
- 需要特定条件触发
- 无 `stats` 字段，只有描述
- 例如：对生命值少于50%的敌人伤害+20%
- 不参与面板计算，用文字记录

## 示例

### 推荐（控制台格式）

```
=== 角色面板 ===
角色: 陈千语 Lv.90
主属性: 敏捷 | 副属性: 力量
武器: 光荣记忆 Lv.90 阶段4 珠子443

=== 基础属性 ===
攻击力: 4190.454
  基础: 297 (角色) + 490 (武器) = 787
  百分比加成: 19.6% (武器)
  能力值加成: 345.2%
  计算: 787 × 1.196 × 4.452 = 4190.454

=== 能力值 ===
敏捷: 590
  171 (角色) + 60 (好感) + 116 (武器) + 243 (装备)
力量: 251
  106 (角色) + 145 (装备)

能力值加成计算:
  敏捷 × 0.5% + 力量 × 0.2%
  = 590 × 0.005 + 251 × 0.002
  = 2.95 + 0.502
  = 345.2%
```

|------|------|
| 攻击力 | 4190.454 |
| 敏捷 | 590 |
