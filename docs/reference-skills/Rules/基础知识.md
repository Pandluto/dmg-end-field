# 基础知识 [高优先级]

> 本文档是高优先级规则源，用于统一核心数值口径和术语定义。

## 定位

本文档只保留两类内容：
- 核心数值规则
- 固定术语定义

本文档不负责：
- 技能入口说明
- 仓库介绍
- 角色或武器导入流程细节

导入步骤请看：
- `Rules/import-character.md`
- `Rules/import-weapon.md`

## 核心数值规则

### 角色面板构成

角色面板由以下三部分构成：
- 角色基础数据：角色自身的等级属性、好感加成
- 武器数据：武器基础攻击、阶段属性、技能词条（含珠子加成）
- 装备数据：装备提供的固定值、百分比或其他词条加成

默认等级：
- **角色默认 90 级**
- **武器默认 90 级**

### 攻击力计算

```text
攻击力= [(干员基础攻击力+ 武器基础攻击力)* (1+ 百分比加成)+ 固定数值加成]* (1+ 能力值加成)
其中 能力值加成= 主能力* 0.005+ 副能力* 0.002
```

其中：
- 角色基础攻击力：角色面板基础值
- 武器基础攻击力：武器面板基础值
- 百分比加成：武器词条、装备词条、珠子或其他明确攻击力百分比加成
- 固定数值加成：装备固定值、珠子固定值或其他明确固定攻击来源
- 能力值加成：由主能力和副能力换算得到的独立乘区
- 默认口径：角色按满好感计算，默认额外提供 `+60` 主能力；如需计算非满好感，可单独覆盖

### 主属性与副属性换算

```text
能力值加成 = 主能力 × 0.005 + 副能力 × 0.002
```

- 主能力每点提供 `0.5%` 能力值加成
- 副能力每点提供 `0.2%` 能力值加成
- 不同角色的主能力/副能力由角色配置决定
- 主能力与副能力必须从四维普通属性中选择，且二者不重复：
  - 力量（strength）
  - 敏捷（agility）
  - 智识（intelligence）
  - 意志（will）

### 暴击期望倍率

```text
暴击期望倍率 = 1 + min(暴击率, 1) × 暴击伤害
```

### 伤害公式

```text
最终伤害 = 攻击力 × 技能倍率
  × (1 + 增伤区)
  × (1 + 易伤区)
  × (1 + 增幅区)
  × (1 + 连击区)
  × (1 + 失衡易伤区)
  × 暴击期望倍率
  × 续航倍率
  ÷ (1 + 防御减免)
  × (1 - 抗性减免)
```

### 武器技能等级计算

```text
武器技能等级 = 武器技能基础等级 + 珠子技能等级
```

限制规则：
- 最终等级不能超过当前阶段的 `maxLevel`
- 无珠子时，珠子技能等级视为 `0`

## 武器相关定义

### 武器构成

武器资料通常由以下部分组成：
- 基础信息
- 攻击成长
- 阶段信息
- 3 个武器技能
- 珠子信息

### 珠子

- 珠子也叫基质
- 珠子对应武器的 3 个技能槽位
- 珠子编码使用三位数字表示三个技能的等级
- 可存在无珠子的情况

默认通用口径：
- 起始编码：`111`
- 满级编码：`663`
- **未提及珠子等级时，默认为 `663`**

示例：
- `111` = 第一个技能 1 级 / 第二个技能 1 级 / 第三个技能 1 级
- `663` = 第一个技能 6 级 / 第二个技能 6 级 / 第三个技能 3 级

### 武器潜力与技能上限

武器潜力影响技能等级上限：
- **0潜武器**：技能等级上限由阶段决定（90级阶段4时为 `663`）
- **满潜武器**：90级时技能等级上限为 `999`（初始 `336` + 珠子 `663`）

计算口径：
- 绝大多数情况下只考虑 **0潜满珠子武器** 或 **满潜满珠子武器**
- 满潜武器的基础等级提升：每个技能 +3 级

### 武器技能类型

武器技能通常分为：
- 属性提升类：直接增加数值，如力量、敏捷、智识、意志
- 百分比提升类：直接增加百分比，如攻击力百分比
- 特殊效果类：包含多段效果、条件触发、持续时间或复合增益

### 源石技艺强度

- 源石技艺强度会影响异常效果和部分技能伤害
- 若具体实现与源码细节冲突，应回查 `src` 中实际计算逻辑后再处理

### 角色配置黑话

六星角色配置常用 `X+Y` 格式表示：

| 格式 | 含义 |
|------|------|
| `0+1某角色` | 角色零潜 + 专武0潜（满珠子） |
| `0+0某角色` | 角色零潜 + 无专武（需指定武器） |
| `6+1某角色` | 角色满潜 + 专武0潜 |
| `6+5某角色` | 角色满潜 + 专武满潜 |

特殊说明：
- **管理员**：默认视为满潜角色
- 六星角色默认按零潜计算，除非明确提及满潜
- 专武默认按0潜满珠子计算，除非明确提及满潜

## 使用原则

- 若用户口述和一般常识冲突，优先按用户当前要求执行
- 若导入 SOP 与本文档冲突，优先以本文档的数值口径和术语定义为准
- 若资料缺失但珠子规则可由基础知识直接确认，可按本文档通用口径补齐
