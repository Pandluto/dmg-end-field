### 数据流向说明

| 数据源 | 用途 | 读取方式 |
|--------|------|----------|
| `sessionStorage.panelSnapshot` | 获取 ATK、暴击率、暴击伤害等基础面板属性 | 只读，不修改 |
| `sessionStorage.infoSnapshot` | 解析出各类型伤害加成百分比 | 只读，不修改 |
| `skillLevelModeMap` | 获取当前技能等级（L9/M3） | 只读，不修改 |
| `已选 Buff (buffList)` | 实时汇总：脆弱区、易伤区、伤害加成区、攻击力乘区 | 不修改 sessionStorage |

### 伤害计算各区数据来源

| 计算分区 | 数据来源 | 说明 |
|----------|----------|------|
| **攻击区** | 实时面板 (panelSnapshot) + 已选Buff | panelSnapshot提供基础ATK，Buff提供攻击力乘区 |
| **暴击区** | 实时面板 (panelSnapshot) + 已选Buff | panelSnapshot提供基础暴击率/暴击伤害，Buff提供暴击加成 |
| **伤害加成区** | 实时面板 (infoSnapshot) + 已选Buff | infoSnapshot解析伤害加成，Buff提供额外加成 |
| **脆弱区** | 已选Buff（独立汇总） | 仅来自Buff，加法叠加后乘法 |
| **易伤区** | 已选Buff（独立汇总） | 仅来自Buff，加法叠加后乘法（不含连击增伤） |
| **连击区** | 已选Buff（独立汇总） | 仅来自Buff，连击增伤独立叠加 |
| **防御区** | 实时面板 (panelSnapshot) | 来自面板防御力计算 |

### 关键约束

1. **sessionStorage 只读**：Buff 计算过程中不得写入 sessionStorage
2. **快照不变更**：panelSnapshot 和 infoSnapshot 在 SkillButton 中始终保持原始值
3. **Buff 实时汇总**：每次 buffList 变化时，重新调用 `calculateBuffTotals()` 计算汇总值
4. **独立计算链**：Buff 加成 → 与面板数据叠加 → 生成实时面板 → 用于伤害计算

## Buff 类型分类

### 1. 伤害倍率区

| Buff 字段 | 说明 | 处理方式 |
|-----------|------|----------|
| `damageBoost` | 伤害倍率提升（加法） | 直接加到最后一个 hit 的倍率上 |
| `damageMultiplier` | 伤害倍率乘倍（乘法） | 直接乘到最后一个 hit 的倍率上 |

**计算规则**：
- 如果多 hit，加到最后一个 hit 上
- 先加后乘：`finalHit = lastHit + damageBoost`，然后 `finalHit = finalHit * damageMultiplier`

### 2. 攻击力乘区

| Buff 字段 | 说明 |
|-----------|------|
| `mainStatBoost` | 主能力提升 |
| `subStatBoost` | 副能力提升 |
| `allStatBoost` | 全能力提升 |
| `atkPercentBoost` | 攻击力百分比提升 |
| `strengthBoost` | 力量属性提升 |
| `agilityBoost` | 敏捷属性提升 |
| `intelligenceBoost` | 智识属性提升 |
| `willBoost` | 意志属性提升 |

**计算规则**（与配置界面规则一致）：

1. **能力值换算**
   ```
   能力值加成 = 主能力' × 0.5% + 副能力' × 0.2%
   主能力' = 主能力 × (1 + mainStatBoost)
   副能力' = 副能力 × (1 + subStatBoost)
   若存在 allStatBoost：主能力' 与副能力' 均继续乘以 (1 + allStatBoost)
   ```

2. **攻击力公式**
   ```
   攻击力 = [(角色基础攻击 + 武器基础攻击) × (1 + atkPercentBoost) + 固定攻击] × (1 + 能力值加成)
   ```

3. **Buff 叠加顺序**
   - 先计算 `mainStatBoost`、`subStatBoost`、`allStatBoost` 得到能力值加成
   - 再应用 `atkPercentBoost` 到基础攻击力
   - 最后乘以能力值加成

### 3. 伤害加成区

| Buff 字段 | 说明 |
|-----------|------|
| `physicalDmgBonus` | 物理伤害加成 |
| `magicDmgBonus` | 法术伤害加成 |
| `fireDmgBonus` | 灼热伤害加成 |
| `electricDmgBonus` | 电磁伤害加成 |
| `iceDmgBonus` | 寒冷伤害加成 |
| `etherDmgBonus` | 自然伤害加成 |
| `allElementDmgBonus` | 全元素伤害加成 |
| `skillDmgBonus` | 战技伤害加成 |
| `chainSkillDmgBonus` | 连携技伤害加成 |
| `ultimateDmgBonus` | 终结技伤害加成 |
| `normalAttackDmgBonus` | 普攻伤害加成 |

**计算规则**：
- 根据干员 `element` 属性区分物理/非物理
- 非物理角色：元素伤害加成 + 法术伤害加成 + 全元素伤害加成
- 物理角色：物理伤害加成 + 全元素伤害加成
- A（普攻）：普攻伤害加成，无所有技能伤害加成
- B/E/Q：对应技能伤害加成 + 所有技能伤害加成

### 4. 暴击区

| Buff 字段 | 说明 |
|-----------|------|
| `critRateBoost` | 暴击率提升 |
| `critDmgBonusBoost` | 暴击伤害提升 |

**数据来源**：实时面板（sessionStorage.panelSnapshot）+ 已选Buff叠加
**计算规则**：直接叠加到面板暴击率/暴击伤害
- 最终暴击率 = panelSnapshot.critRate + Buff汇总.critRateBoost
- 最终暴击伤害 = panelSnapshot.critDmg + Buff汇总.critDmgBonusBoost

### 5. 脆弱区（独立于易伤区）

| Buff 字段 | 说明 |
|-----------|------|
| `physicalFragile` | 物理脆弱 |
| `fireFragile` | 灼热脆弱 |
| `electricFragile` | 电磁脆弱 |
| `iceFragile` | 寒冷脆弱 |
| `etherFragile` | 自然脆弱 |
| `magicFragile` | 法术脆弱 |

**数据来源**：已选Buff（实时汇总）
**计算规则**：
- 加法叠加：`fragileRate = Σ(各元素脆弱)`
- 乘法叠加到最终伤害：`finalDamage = damage * (1 + fragileRate)`

### 6. 易伤区（独立于脆弱区）

| Buff 字段 | 说明 |
|-----------|------|
| `physicalVulnerability` | 物理易伤 |
| `fireVulnerability` | 灼热易伤 |
| `electricVulnerability` | 电磁易伤 |
| `iceVulnerability` | 寒冷易伤 |
| `etherVulnerability` | 自然易伤 |
| `magicTakenDmgBonus` | 法术易伤 |

**数据来源**：已选Buff（实时汇总）
**计算规则**：
- 加法叠加：`vulnerabilityRate = Σ(各元素易伤)`
- 乘法叠加到最终伤害：`finalDamage = damage * (1 + vulnerabilityRate)`
- **注意**：连击增伤(comboDamageBonus)独立为连击区，不在此汇总

### 7. 脆弱区与易伤区的区别

| 特性 | 脆弱区 | 易伤区 |
|------|--------|--------|
| 英文对应 | Fragile | Vulnerability |
| 叠加方式 | 加法叠加后乘法 | 加法叠加后乘法 |
| 独立计算 | 是（独立汇总） | 是（独立汇总） |
| 数据来源 | 已选Buff | 已选Buff |
| 适用场景 | 敌人受到的伤害增加 | 敌人受到的伤害增加 |

### 8. 连击区（独立区域）

| Buff 字段 | 说明 |
|-----------|------|
| `comboDamageBonus` | 连击增伤 |

**数据来源**：已选Buff（独立汇总）
**计算规则**：
- 独立乘法叠加：`finalDamage = damage * (1 + comboDamageBonus)`
- 与脆弱区、易伤区互不影响，独立叠加

**最终伤害公式**：
```
finalDamage = ATK × 倍率 × 暴击期望 × 伤害加成区 × 防御区 × (1 + 脆弱区) × (1 + 易伤区) × (1 + 连击区)
```

### 1. 新建 Buff 计算模块

**文件**：`src/components/CanvasBoard/SkillButtonBuffCalculator.ts`（新建）

将 Buff 汇总计算逻辑独立出来，便于维护和复用。

```typescript

/**
 * 元素类型标签映射
 * 用于显示元素名称
 */

/**

 */
export function calculateBuffTotals(buffs: SkillButtonBuff[]): BuffCalculationResult {

    totalChainSkillDmgBonus: 0,

    totalNormalAttackDmgBonus: 0,
    totalCritRateBoost: 0,
    totalCritDmgBonusBoost: 0,

    totalPhysicalFragile: 0,
    totalFireFragile: 0,
    totalElectricFragile: 0,
    totalIceFragile: 0,

    totalMagicFragile: 0,

    totalPhysicalVulnerability: 0,
    totalFireVulnerability: 0,
    totalElectricVulnerability: 0,
    totalIceVulnerability: 0,
    totalEtherVulnerability: 0,

    totalComboDamageBonus: 0,
    totalDamageBoost: 0,

  // 遍历所有 Buff，累加各字段

    if (buff.atkPercentBoost) result.totalAtkPercentBoost += buff.atkPercentBoost;
    if (buff.mainStatBoost) result.totalMainStatBoost += buff.mainStatBoost;
    if (buff.subStatBoost) result.totalSubStatBoost += buff.subStatBoost;
    if (buff.allStatBoost) result.totalAllStatBoost += buff.allStatBoost;
    if (buff.strengthBoost) result.totalStrengthBoost += buff.strengthBoost;
    if (buff.agilityBoost) result.totalAgilityBoost += buff.agilityBoost;
    if (buff.intelligenceBoost) result.totalIntelligenceBoost += buff.intelligenceBoost;
    if (buff.willBoost) result.totalWillBoost += buff.willBoost;
    // 伤害加成区
    if (buff.physicalDmgBonus) result.totalPhysicalDmgBonus += buff.physicalDmgBonus;
    if (buff.magicDmgBonus) result.totalMagicDmgBonus += buff.magicDmgBonus;
    if (buff.fireDmgBonus) result.totalFireDmgBonus += buff.fireDmgBonus;

    if (buff.iceDmgBonus) result.totalIceDmgBonus += buff.iceDmgBonus;
    if (buff.etherDmgBonus) result.totalEtherDmgBonus += buff.etherDmgBonus;
    if (buff.allElementDmgBonus) result.totalAllElementDmgBonus += buff.allElementDmgBonus;
    if (buff.skillDmgBonus) result.totalSkillDmgBonus += buff.skillDmgBonus;
    if (buff.chainSkillDmgBonus) result.totalChainSkillDmgBonus += buff.chainSkillDmgBonus;
    if (buff.ultimateDmgBonus) result.totalUltimateDmgBonus += buff.ultimateDmgBonus;
    if (buff.normalAttackDmgBonus) result.totalNormalAttackDmgBonus += buff.normalAttackDmgBonus;
    // 暴击区
    if (buff.critRateBoost) result.totalCritRateBoost += buff.critRateBoost;
    if (buff.critDmgBonusBoost) result.totalCritDmgBonusBoost += buff.critDmgBonusBoost;
    // 脆弱区（Fragile）
    if (buff.physicalFragile) result.totalPhysicalFragile += buff.physicalFragile;
    if (buff.fireFragile) result.totalFireFragile += buff.fireFragile;
    if (buff.electricFragile) result.totalElectricFragile += buff.electricFragile;
    if (buff.iceFragile) result.totalIceFragile += buff.iceFragile;
    if (buff.etherFragile) result.totalEtherFragile += buff.etherFragile;
    if (buff.magicFragile) result.totalMagicFragile += buff.magicFragile;
    // 易伤区（Vulnerability）
    if (buff.physicalVulnerability) result.totalPhysicalVulnerability += buff.physicalVulnerability;
    if (buff.fireVulnerability) result.totalFireVulnerability += buff.fireVulnerability;
    if (buff.electricVulnerability) result.totalElectricVulnerability += buff.electricVulnerability;
    if (buff.iceVulnerability) result.totalIceVulnerability += buff.iceVulnerability;
    if (buff.etherVulnerability) result.totalEtherVulnerability += buff.etherVulnerability;
    if (buff.magicTakenDmgBonus) result.totalMagicTakenDmgBonus += buff.magicTakenDmgBonus;
    // 连击区（独立区域）
    if (buff.comboDamageBonus) result.totalComboDamageBonus += buff.comboDamageBonus;
    // 伤害倍率区
    if (buff.damageBoost) result.totalDamageBoost += buff.damageBoost;
    if (buff.damageMultiplier) result.totalDamageMultiplier *= buff.damageMultiplier;
    // 其他
    if (buff.sourceSkillBoost) result.totalSourceSkillBoost += buff.sourceSkillBoost;
  });

  return result;
}

/**
 * 根据干员元素属性和技能类型，计算伤害加成区
 * @param characterElement - 干员元素属性（如 'ice', 'physical'）
 * @param skillType - 技能类型（'A', 'B', 'E', 'Q'）
 * @param parsedDamageBonus - 从 infoSnapshot 解析的伤害加成
 * @param buffTotals - Buff 汇总结果
 * @returns 元素伤害加成值（小数）
 */
export function calculateElementDmgBonus(
  characterElement: string | undefined,
  parsedDamageBonus: Record<string, number>,
  buffTotals: BuffCalculationResult
): number {
  // 判断是否为物理属性
  const isPhysical = characterElement === 'physical';

  if (isPhysical) {
    // 物理角色：物理伤害加成 + Buff 贡献
    return parsedDamageBonus.physicalDmgBonus + buffTotals.PhysicalDmgBonus;
  } else {
    // 非物理角色：对应元素伤害加成 + 法术伤害加成 + 全元素伤害加成 + Buff 贡献
    const elementKey = `${characterElement}DmgBonus`;
    const elementBonus = parsedDamageBonus[elementKey] || 0;
    return elementBonus + buffTotals.MagicDmgBonus + buffTotals.AllElementDmgBonus;
  }
}

/**
 * 根据技能类型，计算技能伤害加成
 * @param skillType - 技能类型
 * @param parsedDamageBonus - 从 infoSnapshot 解析的伤害加成
 * @param buffTotals - Buff 汇总结果
 * @returns 技能伤害加成值（小数）
 */
export function calculateSkillDmgBonus(
  skillType: string,
  parsedDamageBonus: Record<string, number>,
  buffTotals: BuffCalculationResult
): number {
  switch (skillType) {
    case 'A': // 普攻
      return parsedDamageBonus.normalAttackDmgBonus + buffTotals.NormalAttackDmgBonus;
    case 'B': // 战技
      return parsedDamageBonus.skillDmgBonus + buffTotals.SkillDmgBonus;
    case 'E': // 连携技
      return parsedDamageBonus.chainSkillDmgBonus + buffTotals.ChainSkillDmgBonus;
    case 'Q': // 终结技
      return parsedDamageBonus.ultimateDmgBonus + buffTotals.UltimateDmgBonus;
    default:
      return 0;
  }
}

/**
 * 计算所有技能伤害加成（仅 B/E/Q 有效）
 * @param skillType - 技能类型
 * @param parsedDamageBonus - 从 infoSnapshot 解析的伤害加成
 * @param buffTotals - Buff 汇总结果
 * @returns 所有技能伤害加成值（小数）
 */
export function calculateAllSkillBonus(
  skillType: string,
  parsedDamageBonus: Record<string, number>,
  buffTotals: BuffCalculationResult
): number {
  // 普攻无所有技能伤害加成
  if (skillType === 'A') return 0;

  // 其他技能：从战技/连携技/终结技的均值计算
  const avgFromSnapshot = (
    parsedDamageBonus.skillDmgBonus +
    parsedDamageBonus.chainSkillDmgBonus +
    parsedDamageBonus.ultimateDmgBonus
  ) / 3;

  const avgFromBuffs = (
    buffTotals.SkillDmgBonus +
    buffTotals.ChainSkillDmgBonus +
    buffTotals.UltimateDmgBonus
  ) / 3;

  return avgFromSnapshot + avgFromBuffs;
}
```

```typescript
export function getButtonBuffs(buttonId: string): SkillButtonBuff[] {
```

## 验证要点

1. Buff 列表变动时，伤害计算立即重新计算
2. 物理角色使用物理伤害加成，非物理角色使用元素伤害加成 + 法术伤害加成

9. 易伤区乘法叠加：`finalDamage = damage * (1 + vulnerabilityRate)`

13. 不影响 infoSnapshot 和 panelSnapshot
