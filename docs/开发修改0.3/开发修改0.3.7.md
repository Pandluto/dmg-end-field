## 目标

将功能面板改造为多标签页结构，支持切换不同功能模块。

## 目标 DOM 结构

```
#root > div > div > div > div.side-panel > div
  └── .side-panel-content
        ├── .side-panel-header
        │     └── h3 "功能面板"
        └── .side-panel-tabs (新增)
              └── .side-panel-body
                    └── [标签页内容区]
```

## 标签页定义

| 标签名 | Key | 用途 |
|--------|-----|------|
| 伤害加成 | damage | 伤害加成相关配置 |
| 功能1 | function1 | 预留功能区1 |
| 功能2 | function2 | 预留功能区2 |

## 实现步骤

### 1. 创建 `src/components/SidePanel/tabsConfig.ts`

**用途**：存放标签页配置和类型定义

**内容**：
```typescript
// tabsConfig.ts
// 标签页配置 - 集中管理所有标签页定义

/**
 * 标签页定义接口
 */
export interface TabDefinition {
  key: string;       // 标签页唯一标识
  label: string;    // 显示名称
  icon?: string;    // 可选的图标类名
}

/**
 * 可用的标签页列表
 * 按顺序排列
 */
export const SIDE_PANEL_TABS: TabDefinition[] = [
  { key: 'damage', label: '伤害加成' },
  { key: 'function1', label: '功能1' },
  { key: 'function2', label: '功能2' },
];

/**
 * 默认激活的标签页
 */
export const DEFAULT_ACTIVE_TAB = 'damage';
```

### 2. 创建 `src/components/SidePanel/SidePanel.tsx` (重写)

**用途**：SidePanel 根组件，改造为标签页结构

**改动点**：
- 引入 `useState` 管理当前激活的标签页
- 添加 `.side-panel-tabs` 容器包裹标签按钮
- 添加 `.side-panel-tab-content` 容器包裹各标签页内容
- 标签按钮点击切换 activeTab 状态
- 当前激活的标签内容才显示

**结构**：
```tsx
<div className="side-panel">
  <div className="side-panel-content">
    <div className="side-panel-header">
      <h3>功能面板</h3>
    </div>
    <div className="side-panel-tabs">
      {SIDE_PANEL_TABS.map((tab) => (
        <button
          key={tab.key}
          className={`side-panel-tab${activeTab === tab.key ? ' is-active' : ''}`}
          onClick={() => setActiveTab(tab.key)}
        >
          {tab.label}
        </button>
      ))}
    </div>
    <div className="side-panel-body">
      {activeTab === 'damage' && <div className="tab-content-damage">伤害加成内容区</div>}
      {activeTab === 'function1' && <div className="tab-content-function1">功能1内容区</div>}
      {activeTab === 'function2' && <div className="tab-content-function2">功能2内容区</div>}
    </div>
  </div>
</div>
```

### 3. 创建 `src/components/SidePanel/SidePanel.css` (新增样式)

**用途**：标签页样式

**样式定义**：
- `.side-panel-tabs` - 标签按钮容器，flex 布局
- `.side-panel-tab` - 单个标签按钮
- `.side-panel-tab.is-active` - 激活状态的标签
- `.side-panel-tab-content` - 标签页内容容器
- `.tab-content-damage` / `.tab-content-function1` / `.tab-content-function2` - 各标签内容区

### 4. 导出更新

确保 `SidePanel/index.ts` 正确导出重命名后的组件。
