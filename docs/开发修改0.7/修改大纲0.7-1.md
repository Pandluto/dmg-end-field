# 修改大纲0.7-1

[阶段目标]
- 本阶段目标是在双击技能按钮后的并排弹窗体系内，新增 `异常伤害` 弹窗。
- `异常伤害` 不做成右侧 `ToolPanel` 新 tab，不并入现有 `DamageTab.tsx`。
- 本阶段不把异常体系伪装成普通 Buff；要建立“普通 Buff”和“异常项”并存的独立结构。
- 本阶段优先打通主链路：`选择异常项 -> 生成异常状态/异常伤害 -> 接入当前按钮伤害计算展示`。

[核心结论]
1. `异常伤害` 不是普通 Buff 面板
   - 它是技能按钮弹窗内的独立子系统。
   - 它既要管理异常状态，也要管理异常伤害。

2. 异常项必须拆成两类
   - `异常状态`
     - 例：
       - 导电
       - 腐蚀
       - 碎甲
     - 特点：
       - 不是普通 Buff
       - 需要来源角色
       - 数值来自来源角色面板缓存
       - 主要影响后续伤害区
   - `异常伤害`
     - 例：
       - 猛击
       - 碎冰
       - 倒地
       - 击飞
       - 冻结初始伤害
       - 燃烧初始伤害
       - 法术爆发
     - 特点：
       - 是独立结算 hit
       - 不伪装成普通 Buff
       - 默认由当前按钮角色触发

3. 第一版不做全局“场上异常状态表”
   - 不做自动同步全队共享状态。
   - 先做“当前按钮主动选择要吃谁的异常状态”。
   - 即：
     - 当前按钮打开 `异常伤害` 弹窗
     - 勾选 `导电/碎甲/腐蚀`
     - 展开 `来源角色`
     - 选择来源角色
     - 读取来源角色面板缓存计算异常数值
     - 将该异常状态作用到当前按钮的伤害展示

4. 第一版不把异常状态混入普通 Buff 主表
   - `ddd.all-buff-list.v1` 继续只存普通 Buff 实体。
   - 异常状态和异常伤害必须有独立存储和独立类型。
   - UI 可显示得像 Buff 卡片，但底层不得复用 `SkillButtonBuff` 主结构。

[来源角色规则]
1. 仅状态型异常支持 `来源角色`
   - 适用：
     - 导电
     - 腐蚀
     - 碎甲
   - 不适用：
     - 猛击
     - 碎冰
     - 法术爆发
     - 倒地
     - 击飞

2. `来源角色` 为显式选择
   - 当前按钮不自动推断来源。
   - 用户手动选：
     - 自己
     - 队友 A
     - 队友 B
     - 队友 C

3. `来源角色` 单选
   - 同一个 `导电` 选择项只允许一个来源实例。
   - 不允许同时挂 `导电-来源A` 和 `导电-来源B`。

4. 读取来源数据
   - 读取来源角色的缓存：
     - `panelSnapshot`
     - `infoSnap`
     - 装备面板中的源石技艺强度
   - 如果当前面板缓存没有独立 `源石技艺强度` 字段，本阶段必须先补这个字段进入角色快照链路。

[计算规则要求]
1. 异常状态和异常伤害分算
   - `anomalyEffectCalculator.ts`
     - 导电
     - 腐蚀
     - 碎甲
   - `anomalyDamageCalculator.ts`
     - 猛击
     - 碎冰
     - 倒地
     - 击飞
     - 燃烧初始
     - 燃烧持续
     - 冻结初始
     - 法术爆发

2. 必须显式编码的规则
   - 导电/腐蚀/燃烧初始/冻结：
     - `80% * (1 + 异常等级)`
   - 燃烧持续：
     - `12% * (1 + 异常等级)`
   - 碎冰：
     - `120% * (1 + 异常等级)`
   - 法术爆发：
     - `160%`
   - 倒地/击飞：
     - `120%`
   - 碎甲：
     - `50% * (1 + 异常等级)`
   - 猛击：
     - `150% * (1 + 异常等级)`

3. 等级系数区
   - 法术异常 / 法术爆发：
     - `1 + (level - 1) / 196`
   - 物理异常：
     - `1 + (level - 1) / 392`
   - 碎冰特殊规则：
     - 伤害类型是物理
     - 但算法按法术异常系数走

4. 源石技艺强度区
   - 伤害乘区：
     - `1 + 源石技艺强度 / 100`
   - 附带效果增强：
     - `2 * 源石技艺强度 / (源石技艺强度 + 300)`

5. 附带效果计算
   - 导电：
     - 法术伤害提高 `12/16/20/24%`
     - 再乘附带效果增强
   - 碎甲：
     - 物理伤害提高 `12/16/20/24%`
     - 再乘附带效果增强
   - 腐蚀：
     - 初始降抗
     - 每秒降抗
     - 上限降抗
     - 都要乘附带效果增强

6. 持续段计入总伤
   - `燃烧持续` 必须支持开关：
     - 是否计入总伤
   - 结果区至少显示：
     - 单次伤害
     - 持续段伤害
     - 总伤
