打包版快速上手

1. 打开工具

1.1 双击 exe。
1.2 看到 Shell 窗口后，点“打开浏览器 Web”。
1.3 进入 Web 主界面后，主要操作都在这里完成。


2. 应用数据

2.1 在 Shell 左侧点“数据”。
2.2 在存档列表里点一个存档。
2.3 勾选要应用的内容，比如干员、武器、装备、Buff、排轴。
2.4 点“应用这个存档”。
2.5 回 Web 主界面刷新。

备份当前数据：
2.6 在“保存当前数据”里填名字。
2.7 点“保存到本机”或“保存到共享”。


3. 下载图片

3.1 在 Shell 左侧点“图片”。
3.2 点“检查更新”。
3.3 有更新就点“下载并切换”。
3.4 下载完成后回 Web 主界面刷新。

零散图片：
3.5 回 Web 主界面。
3.6 点底部“图片管理”。
3.7 点“导入”。


4. 选择干员

4.1 在主界面先选干员。
4.2 最多选 4 个。
4.3 选完点“开始排轴”。
4.4 如果没有干员，先点底部“编辑干员”。


5. 配置干员

5.1 点底部“角色配置”，或顶部抽屉里的“干员配置”。
5.2 用右侧头像切换当前干员。
5.3 调干员等级、潜能、主能力、副能力。
5.4 点武器区域，选择武器。
5.5 调武器等级、潜能、效果等级。
5.6 点装备槽，选择装备。
5.7 调装备词条等级。
5.8 调 A/B/E/Q 技能等级。
5.9 配完回排轴。


6. 排轴

6.1 右边是技能列表，左边是时间轴。
6.2 把右边技能拖到左边格子里。
6.3 拖动技能节点可以改位置。
6.4 左键点技能节点，看伤害详情。
6.5 左键按住技能节点，可以拖动位置。
6.6 右键技能节点，可以删除、复制、换技能。
6.7 底部“保存”“恢复”“分享”可以管理排轴。


7. 单个技能上 Buff

7.1 点一个技能节点，进入技能详情。
7.2 右上角点“+”，或者直接按 Tab 打开 Buff 面板。
7.3 Buff 面板打开后，按 Tab 切换入口。
7.4 可切到干员、武器、装备、通用 Buff、异常状态等入口。
7.5 输入关键词搜索。
7.6 左键点 Buff 加入。
7.7 Esc 关闭 Buff 面板。
7.8 在 Buff 列表里可以删除、禁用、启用、加减层数。
7.9 右键 Buff 可以快速移除。


8. 微调伤害

8.1 展开“Hit 微调”，可以单段开关 Buff、调层数。
8.2 点准星按钮，可以改目标抗性。
8.3 点锁按钮，可以防止节点误拖。
8.4 看右侧计算过程，可以确认 Buff 进了哪个乘区。


9. 批量 Buff

9.1 打开顶部抽屉。
9.2 点“BUFF批量操作”。
9.3 左键点技能节点，选中或取消选中。
9.4 点“+”批量增加 Buff。
9.5 按 Tab 打开候选 Buff 面板。
9.6 候选面板打开后，按 Tab 切换入口。
9.7 输入关键词搜索。
9.8 左键点 Buff，加入本轮候选。
9.9 左键点目标节点。
9.10 点确认。

批量删除：
9.11 点“-”。
9.12 选要删除的 Buff 和目标节点。
9.13 点确认。

其他：
9.14 点框选按钮后，左键拖框可以批量选节点。
9.15 Esc 可以关闭候选 Buff 面板。
9.16 漏斗可以按已有 Buff 筛节点。
9.17 编辑目录可以批量补齐或移除共同 Buff。


10. 自定义数据

10.1 点底部“编辑干员”，新增或修改干员。
10.2 点底部“编辑BUFF”，新增或修改通用 Buff。
10.3 点底部“编辑武器”，新增或修改武器。
10.4 点底部“编辑装备”，新增或修改装备。
10.5 点底部“图片管理”，导入和管理图片。
10.6 改完后回主界面刷新，或重新选干员。


11. AI CLI

11.1 点底部“AI CLI”。
11.2 输入 help 看命令。
11.3 输入 fill.task，把任务要求给 AI。
11.4 让 AI 只返回 JSON。
11.5 用 fill.check 检查 JSON。
11.6 用 fill.apply 生成提案。
11.7 按 Y 批准。
11.8 再按 Y 保存。

注意：
fill.apply 只是生成提案，不是直接保存。
