# 开发修改 1.4.0 - 四线谱组数扩展

## 约束
- 本计划仅针对四线谱组数扩展功能
- 不修改技能按钮拖拽逻辑
- 不修改角色规则
- 配置参数需保持向后兼容

## 修改需求

### 1. 增加谱线组数
- 当前：2组四线谱
- 目标：3-5组（支持动态调整）
- 渲染策略：使用垂直滚动

### 2. 谱线组数控制按钮
- 添加「+」「-」按钮控制组数
- 最少2组
- 最多5组
- 位置：工具栏

### 3. 组间间距调整
- 第一组和第二组之间增加明显间隔
- 后续组之间保持均匀间距

## 技术方案

### 配置变更
```typescript
// types/index.ts
staffCount: number;  // 动态 2-5，默认2
groupSpacing?: number;  // 组间额外间距，默认60
```

### 布局计算修改
```typescript
// utils/layout.ts
// calculateLineY 需要考虑组间间距
```

### 组件修改
```typescript
// CanvasBoard/index.tsx
// 1. 添加工具栏按钮

```

### 渲染结构
```
.canvas-container
  overflow-y: auto
  .canvas
    渲染 staffCount 组谱线
    每组 4 条线 + 组间间距
```

## 实现步骤

1. 修改 `types/index.ts` - 添加 staffCount 动态支持和 groupSpacing
2. 修改 `utils/layout.ts` - calculateLineY 支持组间距
3. 修改 `CanvasBoard/index.tsx`:
   - 添加 state 管理 staffCount
   - 添加工具栏 +/- 按钮
   - 容器支持垂直滚动
4. 测试验证

- [ ] 能添加/删除谱线组（2-5组范围）
