# 开发修改 1.4.7 - 缩短谱线右侧长度

## 约束
- 本计划仅针对谱线右侧渲染过长问题
- 不修改节点数量和间距
- 不修改画布宽度计算
- 仅调整 CSS `right` 属性值

## 问题分析

### Bug: 谱线右侧渲染过长

**表现：**
- 谱线使用 `right: 0` 拉伸到右侧
- 当节点数量较多时，谱线超出最后一个节点

**根因：**
`.staff-line` CSS：
```css
.staff-line {
  position: absolute;
  left: 80px;
  right: 0;  /* 拉伸到右侧，可能超出节点 */
}
```

## 技术方案

### CSS 方案

缩短 `.staff-line` 的 `right` 值：
```css
.staff-line {
  position: absolute;
  left: 80px;
  right: 20px;  /* 从 0 改为 20px */
}
```

**计算依据：**
- 节点宽度 8px，最后节点 X = 80 + 39 * 22 = 938px
- 节点右侧边界 = 938 + 8 = 946px
- 谱线 `right: 20px` 可确保不超出画布

## 实现步骤

1. 修改 `CanvasBoard.css` 中 `.staff-line` 的 `right` 值
2. 测试验证谱线不超过最右侧节点

## 验收标准
- [ ] 谱线右侧不超过最后一个节点
- [ ] 谱线与节点对齐美观
