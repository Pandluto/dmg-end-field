# 开发修改 1.4.8 - 缩小谱线组之间 Y 轴间距

## 约束
- 本计划仅针对谱线组间距调整
- 不修改谱线组数量控制逻辑
- 不修改谱线组内部布局
- 仅调整 `groupSpacing` 配置值

## 问题分析

### 需求：缩小谱线组之间 Y 轴间距

**表现：**
- 相邻谱线组之间间距过大，占用屏幕空间
- 需要将间距缩小一半以节省垂直空间

**根因：**
`DEFAULT_CANVAS_CONFIG` 中 `groupSpacing: 80` 数值较大

## 技术方案

### 配置调整

修改 `types/index.ts` 中 `DEFAULT_CANVAS_CONFIG.groupSpacing`：

```typescript
// 修改前
groupSpacing: 80,

// 修改后
groupSpacing: 40,
```

**计算依据：**
- 当前间距 80px
- 缩小一半 = 40px
- 2 组时总高度：2 * 300 + 1 * 40 = 640px（原 680px）
- 5 组时总高度：5 * 300 + 4 * 40 = 1660px（原 1720px）

## 实现步骤

1. 修改 `types/index.ts` 中 `groupSpacing` 从 80 改为 40
2. 测试验证 2-5 组谱线显示正常
3. 测试验证拖拽、吸附功能正常

## 验收标准
- [ ] 谱线组之间间距缩小为原来的一半
- [ ] 2-5 组谱线都能正常显示
- [ ] 拖拽、吸附功能正常
