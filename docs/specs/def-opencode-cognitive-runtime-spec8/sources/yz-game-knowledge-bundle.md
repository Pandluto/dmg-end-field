# game-knowledge

Use this skill when the user asks about 终末地 game mechanics, team compositions, character builds, rotation guides, equipment recommendations, or player slang/terminology.

## Knowledge Architecture

This skill distills game knowledge from 月咒's strategy video transcripts, organized into five layers (inspired by 女娲.skill's distillation methodology):

| Layer | Content | Files |
|-------|---------|-------|
| **怎么说话** | Player slang, nicknames, ASR error patterns | `references/glossary.md` |
| **怎么打** | Rotation cycles, burst sequences, skill combos | 10 team guide files |
| **怎么配** | Equipment sets, weapon priority, stat thresholds | Each guide's equipment section |
| **怎么判断** | Hot-start vs cold-start, boss-specific adaptations | Each guide's tactics section |
| **什么不换** | Core characters that cannot be substituted | Each guide's team overview |

## Trust Order

1. DEF typed tools (`def.workbench.*`, `def.character.resolve`, `def.buff.resolve`, etc.) for real-time game state.
2. This skill's `references/` for verified player-community knowledge.
3. External game databases (wiki, AKEDatabase) as fallback.

## Core Rules

- When the user asks about team building, rotation, or equipment, consult the relevant guide in `references/` first.
- Use `glossary.md` to resolve player nicknames (e.g., "42" → 莱万汀/史尔特尔, "小羊" → 艾尔黛拉) and correct common ASR errors before searching.
- Equipment names: note that "三X" means "3 pieces of X set" (e.g., "三动火" = 3 pieces of 动火用). Search for the set name without the number prefix.
- Character names: the streamer may use nicknames or mispronunciations. Cross-reference with `glossary.md` for the official in-game names.
- Do not hardcode character stats, skill multipliers, or equipment values from the guides — they reflect a specific game version and may be outdated. Use DEF typed tools for live data.
- Guide content is advisory — the user may have different roster, resources, or preferences. Present options, not absolutes.

## 一、秋栗 vs 安塔尔

上个视频讲的是火队带安塔尔的打法，这期补一个带秋栗的攻略。

| | 安塔尔 | 秋栗 |
|------|------|------|
| 偏向 | 对单 | **对群** |
| 战技 | 60 秒脆弱，只能挂一个目标 | 小范围群体火附着 |
| 启动 | 常规 | **更快**，挂火更舒服 |
| 大招 | — | 点天赋后触发**连击效果** |

### 秋栗连击效果

秋栗开大后，下一个战技或大招获得额外伤害加成：

| 下个技能 | 伤害提升 |
|----------|---------|
| 战技 | +30% |
| 大招 | +20% |

> 这个伤害加成是**纯净乘区**（独立乘区）。秋栗开大后，莱万汀接强化战技可以直接额外 +30% 伤害，爆发非常香。

在对群环境下，秋栗战技 + 狼卫连携（也是群体火附着）可以很快给小怪群同时挂上两层灼热附着 → 莱万汀更高频率触发强化战技 → 更轻松叠四层熔火 → 更快开出大招。

## 二、冷启动教学

大招充能是两套一模一样的连招，记住一套就行。

### 第一波充能轴

1. **秋栗**的战技 + **狼卫**的战技 → boss 挂**两层灼热附着**。
2. 触发狼卫连携技 → 挂**第三层火**。
3. **莱万汀**打重击吸收三层灼热附着 → **三层熔火**（花瓣三个白）。
4. 艾尔黛拉连携延迟爆炸后 → 触发莱万汀连携技 → 使用后闪避取消后摇 → +1 层 → **四层熔火**。
5. 满熔火 → 莱万汀**强化战技**消耗熔火，增加伤害和恢复大招能量。

---

### 第二波充能轴

> 启动前看左下角狼卫连携冷却（蓝条上的白条满了 = 好了）。如果打狼卫三火连招可节省秋栗 100 技力，但刚需狼卫五潜 + 手速。

本轮需要加入艾尔黛拉战技挂脆弱，其他和第一波一样。

1. **秋栗**战技 + **狼卫**战技 → 两层灼热附着。
2. 狼卫连携技 → 第三层火。
3. 莱万汀重击吸收三层火 → 三层熔火。
4. 触发艾尔黛拉连携技 → 触发莱万汀连携技 → +1 层熔火 → 四层熔火。
5. **艾尔黛拉**战技把腐蚀转为脆弱。

**此时分情况**：

| 配置 | 下一步 |
|------|--------|
| 零潜 + 充能衣 / 满潜 | 1 次强化战技 → 大招充能完毕（2 强化 + 2 连携） |
| 零潜 + 生物辅助面板（无充能） | 等莱万汀连携技转好 → 打强化战技 → 共 2 强化 + 3 连携充满 |

---

### 全员开大爆发

boss 身上有艾尔黛拉脆弱 + 专武异常 → 技力充足时，莱万汀强化战技 + 连携技充满大招。

1. **全员开大**：莱万汀 → 狼卫（打失衡） → 艾尔黛拉（打失衡）。
2. 按住左键不松手，连携亮了就按。
3. 四层熔火时 → **秋栗开大**触发连击效果。
4. 莱万汀**强化战技**（吃连击 +30% 伤害） → 莱万汀连携技。
5. 继续循环。斩杀判断：
   - 能斩杀 → 大招期间打强化战技
   - 不能 → 留四层熔火，等大招结束后打强化战技充能

## 三、热启动

热启动只需要打一个爆发前置，就可以无脑连招。

### 前置

1. 主控打重击 → 触发**艾尔黛拉**连携技，挂腐蚀。
2. **狼卫三火连招**：战技 → 连携（后空翻） → 落地前开大招打断 → 大招结束后再次触发连携。

> 需要狼卫五潜。没满潜也可以打，就是少一层火。

3. **艾尔黛拉**战技把腐蚀转脆弱。
4. 打重击把火吸到手里。

### 爆发

1. 莱万汀直接开大（手里四层熔火）。
2. **秋栗**开大触发连击效果。
3. 莱万汀强化战技（吃连击 +30%）。
4. 按住左键 → 连携好了就按 → 四层熔火打强化战技 → 循环。

## 四、进阶技巧

### 技巧 1：补充能

大招差一丝充能时：艾尔黛拉连携挂腐蚀 / 狼卫大招挂燃烧 / 莱万汀强化战技 → 均可触发莱万汀连携技。

狼卫战技 → 狼卫连携技（后空翻）→ 落地前开狼卫大招 → 再次触发连携

## 一、阵容概述

大家好，我是月咒，专注终末地战斗攻略的主播。

今天带来的阵容是以弭弗作为主 C 的**传统物理队**。这套队伍是目前弭弗所在的队伍里，给他提升拐力最多、伤害占比最高的一套，也是最能体现弭弗爽感的队伍——哪怕是零潜弭弗，也能明显感觉到两拳下去伤害有多爆炸。

## 二、热启动排轴教学

> 使用 0+1 弭弗演示，伤害可观。一共打三波伤害：四碎甲 → 猛击 → 四猛击。

### 2.1 第一波：四碎甲轴

**起手骏卫直接开大**。由于是热启动全员满大，骏卫大招每挂一层破防都会恢复技力，所以一定是先开的——早用早 CD。

1. **黎风**的战技给 boss 挂物理脆弱，并添加**第 1 层破防**。
2. 触发**陈千语的连携技**。
3. 主控打一个**重击**触发黎风的连携技，直接使用。
4. **陈千语的连携**给 boss 挂**第 2 层破防**。
5. **黎风的连携**获得**连携 buff**。
6. **黎风直接开大**——大招吃掉连携 buff，获得 20% 伤害提升。黎风大招分为两段，每一段都能叠一层破防。
7. 黎风开大过程中，穿插**弭弗的连携技**给 boss 挂物理脆弱。
8. boss 身上**四层破防**后，立即用**骏卫的战技引爆**四层破防，给 boss 挂**碎甲 buff**。
9. 用**骏卫的连携技**收尾，恢复一点技力。

> 四碎甲轴完成。

---

### 2.2 第二波：猛击轴

四碎甲轴后需要重新给 boss 挂四层破防。

1. **黎风**的战技挂**第 1 层破防**。

> 弭弗二段战技虽然也能挂一层破防而且只要 50 技力，但会让黎风的大招充能阈值变高（充能阈值会在后面配装里讲）。

2. 等陈千语连携转好，用**陈千语的战技**挂**第 2 层破防**。
3. **陈千语的连携**挂**第 3 层破防**。
4. **弭弗的大招**挂**第 4 层破防**。

> 追求极限伤害：可以先用主控打重击触发黎风的连携拿到连携 buff → 陈千语大招吃连携打出更高伤害，同时还能降低黎风的充能压力。不过这里选择了直接爆发，操作更连贯、手感更好。

5. **陈千语直接开大**打输出。
6. **弭弗**接二段战技，用**猛击爆掉**四层破防，再接三段战技打出追加伤害。

> 这轮猛击伤害打完以后，还要在 boss 的四层碎甲 buff 消失之前，再打出一次四层破防猛击——这样才能把碎甲 buff 的收益吃满。

---

### 2.3 第三波：四猛击轴（斩杀）

1. **黎风**的战技挂**第 1 层破防**和物理脆弱。
2. 触发**陈千语的连携**。
3. 主控打重击触发**黎风的连携**，两个连携同时使用 → boss 身上**两层破防**，获得连携 buff。
4. **黎风开大**给 boss 叠至**四层破防**。
5. 触发并使用**弭弗的连携技**，解锁二段战技。
6. **弭弗二段战技**打出猛击，爆掉四层破防。
7. **弭弗三段战技**追加伤害，完成斩杀。

---

### 2.4 黎风双大打法 vs 三破防斩杀

**双大打法充能阈值**：

| 潜能 | 充能阈值 |
|------|---------|
| 零潜 | 211.76% |
| 满潜 | 180% |

如果不想堆这么高充能，第三波斩杀阶段可以改为**三破防满级大法**：

1. 黎风战技挂第 1 层破防。
2. 主控重击。
3. 陈千语和黎风的连携技。
4. 陈千语战技挂第 3 层破防。
5. 弭弗连携技挂物理脆弱，解锁二段战技。
6. 弭弗二段战技接三段打出爆发伤害。

- **零潜**：充能阈值约 211.76%，装备精炼满充能也不太够 → 需要额外带充能武器。武器在**使命必达**和**嵌合正义**里二选一。带了充能武器后，装备不需要精炼充能。
- **满潜**：充能阈值约 180%，全身充能装备即可，不需要精炼。武器可选输出武器，比如专武、**负山**或**晓勇**（待确认）。

#### 方案 B：点剑输出装

- 如果黎风养成全部拉满（战技专三、珠子全满、装备精炼跟上），在这套队伍里伤害占比大概在 20% 左右，有一定输出能力。
- **注意**：穿输出装后没有办法打双大循环。
- **优势**：黎风自身伤害和打失衡的能力更强。
- **建议**：高养成度（专三满、珠子全满、精炼全满）非常推荐穿这套。

### 3.4 骏卫

- **技能加点**：大招和战技专一即可。

# 【懒人配队】弭弗x洛茜x洁尔佩塔x佩丽卡 公式化排轴攻略&装备养成推荐

>
> 阵容：弭弗 / 洛茜 / 洁尔佩塔 / 佩丽卡
>
> 类型：洛茜 3+1 体系变种，懒人公式化排轴。

---

## 一、阵容定位

大家好，我是月咒，专注终末地战斗攻略的主播。

这套阵容是**洛茜 3+1 体系**的一个变种——以洛茜 + 洁尔佩塔 + 佩丽卡为核心，第四个位置换成弭弗。之前做过两套类似阵容，第四人分别是汤汤和管理员。

### 弭弗 vs 汤汤

| | 汤汤 | 弭弗 |
|------|------|------|
| 群攻能力 | 更强，弥补洛茜群攻短板 | 一般 |
| 操作 | 按键次数更少，更简单 | 洛茜连携二段 + 弭弗多段战技，按键多，操作感和打击感更强 |
| 爆发力 | 一般 | 更强，短时间内打出更高爆发 |

### 弭弗 vs 管理员

- 管理员：全队 +15% 攻击力，有一定拐力
- 弭弗：自身伤害更高。这套队伍里弭弗能吃到的拐力主要是佩丽卡的导电易伤，队伍主要输出还是洛茜（两个辅助都属于法术拐）。

**伤害比例**：弭弗约 40%，洛茜约 60%。

只看一波热启动爆发，洛茜伤害更高。但热启动后的充能阶段洛茜基本没太多伤害，而弭弗依然可以打出一次四层破防猛击 + 三段战技追加伤害。洛茜主要在爆发阶段打高伤害，弭弗则在爆发和充能阶段都能补可观伤害。

---

## 二、热启动排轴教学

> 热启动全员满大。洛茜和弭弗谁主控都可以。

### 2.1 第一波爆发

1. **佩丽卡**战技，给 boss 挂**电磁附着**。
2. 主控打**重击**，触发并使用佩丽卡的连携技。
3. 接着触发**洁尔佩塔的连携技**——按完以后**一定要第一时间接洛茜的战技**，给 boss 挂**爪印斫痕** buff。
4. **洛茜**一段连携 + 二段连携，触发暴击 buff，把 boss 破防层数叠到 **4 层**。
5. **佩丽卡**开大——导电易伤，全队加攻。
6. **洁尔佩塔**开大——高额脆弱。
7. **洛茜**开大直接爆发。
8. 洛茜开大过程中，使用**弭弗的连携技**给 boss 挂物理脆弱，同时战技变为二段。
9. 弭弗**二段战技** → **三段战技**追加伤害收尾。

---

### 2.2 第二波充能轴

> 尽可能技力两格时启动，输出更流畅。技能顺序和第一轮只有细微区别。

1. **佩丽卡**战技挂电磁附着。
2. 主控重击 → 佩丽卡连携技。
3. **洁尔佩塔**的连携技。
4. **洛茜**一段连携 + 二段连携，触发暴击 buff、挂破防 → boss **三层破防**。
5. **弭弗**开大招，给 boss 挂**第四层破防**。
6. **弭弗**连携技挂物理脆弱 → 二段战技 → 三段战技追加伤害直接斩杀。

> 如果 boss 没死，连携可以保留到下一轮释放。

---

## 三、冷启动排轴教学

冷启动和热启动基本一样。

### 第一轮充能

1. **佩丽卡**战技挂电磁附着。
2. 主控重击 → 佩丽卡连携技。
3. **洁尔佩塔**连携技——用出去的瞬间**立刻衔接洛茜的战技**挂爪印斫痕 buff。
4. **洛茜**连携一段 + 连携二段 → boss 四层破防。
5. 触发并使用**弭弗连携技**挂物理脆弱 → 二段战技 → 三段战技追加伤害。

> 第一轮充能完成。

### 第二轮爆发轴

和热启动第一波一样。尽可能两格技力启动。

1. **佩丽卡**战技挂电磁附着 → 主控重击 → 佩丽卡连携技。
2. **洁尔佩塔**连携技——**第一时间**接洛茜战技挂爪印斫痕。

> 为什么说第一时间：洛茜开大可以吃到佩丽卡的导电易伤。如果进慢了，洛茜大招会少伤害。

3. **洛茜**连携一段 + 二段 → boss 四层破防。
4. **佩丽卡**开大，**洁尔佩塔**开大。
5. **洛茜**开大。
6. **弭弗**连携技挂物理脆弱 → 二段战技猛击爆掉破防 → 三段战技追加伤害，完成斩杀。

# 【终末地】卡缪 超详细养成配队指南 角色定位、提升分析、全员装备推荐

>
> 涵盖：角色定位 / 连击机制 / 队伍提升分析 / 配队推荐 / 全员装备养成

---

## 一、角色定位

对于卡缪，先说结论：**他不是主 C，而是一个功能性先锋**。目前主要服务于两套体系：

- **42 火队**（莱万汀体系）
- **洛西物理队 & 混伤队**

核心作用就三点：**回技力、挂灼热附着、提供连携 buff**。

---

## 二、连击机制详解

连携 buff 的作用是**强化下一次关键技能**：

| 技能类型 | 伤害提升 |
|----------|---------|
| 战技 | +30% |
| 终结技 | +20% |

> 连携 buff 属于**独立乘区**，实际收益非常高。

连携 buff 可以叠层，最高 4 层：

| 层数 | 战技伤害提升 | 终结技伤害提升 |
|------|-------------|---------------|
| 1 层 | 30% | 20% |
| 2 层 | 45% | 30% |
| 3 层 | 60% | 40% |
| 4 层 | 75% | 50% |

> 实战中一般主要用 1 层连携。**42 火队**里连携给 42 的强化战技吃；**洛西队**里给洛西的大招吃。

---

## 三、队伍提升分析（卡缪 vs 旧火队）

以前火队常用**秋栗**，但秋栗有一个问题：它的连携更依赖**失衡节点**触发。如果遇到没有多失衡节点的 boss，连携触发变少 → 大招循环变慢 → 技力回复不够稳定 → 连携触发少。

**卡缪的优势**：

- 战技可以挂灼热附着，还有**传递机制**——一旦机制跑起来，一个技能就能带动多次灼热附着收益
- 火队和洛西队实战中能更频繁地吸收灼热附着来触发卡缪的连携 → 触发天赋连携 buff → 技力恢复次数变多
- 连携触发次数增加 → 卡缪大招回转更快
- 开完终结技后，还会**额外再给一次强化连携技**——同样可以触发连携 buff 并恢复技力

**总结**：卡缪可以大幅度提升火队的战斗力。在 42 火队和洛西队里替代秋栗后，整体拐力和技力都有大幅提升。

- 如果之前玩火队很少用秋栗的连携 → 提升感知非常明显
- 如果之前比较熟练、经常能用到秋栗的连携 → 不一定有明显数字感知，但一定能感受到**自由控制连携时机**的体验提升

> **抽取建议**：
> - 42 火队玩家 → 卡缪本体优先级很高
> - 洛西队玩家 → 有抽取价值（洛西物理队需要法术副手位）
> - 两个体系都不玩 → 看自己的资源和后续规划

---

## 四、配队推荐

### 4.1 42 火队

**核心阵容**：莱万汀 + 卡缪 + 狼卫 + 第四人

第四人的选择决定整队的风格：

#### A. 艾尔黛拉（小羊）—— 对单最稳定

- 战技提供长时间高数值脆弱效果
- 同时触发专武异常增伤
- 连携技提供腐蚀，并可以触发 42 的连携 → 更快叠出熔火层数
- 整体非常均衡：有拐力也有生存

> **推荐**：最简单、最稳定，综合强度非常高。

#### B. 洁尔佩塔（安洁莉娜 / 杰哥 / 洁哥）—— 群怪 & 爆发

- 优势是聚怪能力强，可以把敌人集中起来
- 火队竞速打法靠洁尔佩塔大招 + 卡缪连携打短时间爆发
- 卡缪代替秋栗后，这套就是之前的升级版

#### C. 安塔尔 —— 省心

- 提供一定拐力
- 每轮循环都能补一层灼热附着
- 优先级比小羊低一点，但体验也不错

---

### 4.2 洛西物理队

**基础阵容**：洛西 + 卡缪 + 猛击手 + 自由位

**固定位**：洛西 + 卡缪（不变）

**猛击手**：优先级最高的是**弭弗**（负责破防收尾和爆发输出）。没有弭弗的话可用**管理员**或**大潘**替代，整体思路一样。

**自由位**：

#### A. 骏卫 —— 高节奏操作型

- 骏卫是先锋，提供碎甲效果
- 配合专武或热熔切割器提供额外增益
- 加上骏卫和卡缪的回费能力 → 技能循环非常活跃、节奏快、操作感强
- 适合喜欢操作、手感强、节奏快的玩家

#### B. 余烬 —— 稳定生存型

- 连携不仅能提供治疗，还能挂一层破防
- 如果破防触发在关键节点，相当于直接省掉一次技能成本
- 适合不想紧张操作或打高压环境的玩家

#### C. 陈芊羽 —— 简单易上手

- 缺点：队伍生存稍弱
- 好理解好上手，但容错一般

#### D. 黎锋

- 主要作用：三点战技提供物理脆弱和破防，大招可补两层破防，连携能触发连携 buff
- 但弭弗的战技猛击伤害不吃连携 buff，收益最大的是洛西大招
- 卡缪已经提供连携 buff 了
- **能用，但不是最优解**

---

### 4.3 洛西混伤队

**阵容**：洛西 + 卡缪 + 佩丽卡 + 洁尔佩塔

- 卡缪的位置以前一般由汤汤或弭弗担任，现在换成卡缪完全可行
- 适合喜欢打一波高爆发伤害的玩家

**循环思路**：
1. 佩丽卡连携提供导电和法术增伤
2. 洁尔佩塔提供破防和高额脆弱
3. 卡缪提供连携 buff 和技力恢复
4. 三个增益集中到洛西身上 → 洛西用连携接大招打出高额爆发

> 和之前带汤汤/弭弗的阵容比，差别不大。危机活跃以外的环境都能轻松应对。汤汤 + 洛西操作更简单无脑一些，但这套也不算难。

# 【终末地】弭弗·首发攻略 定位、技能、手法、配队、武器装备养成建议

>
> 弭弗首发全面攻略：定位分析 / 技能拆解 / 输出手法 / 配队推荐 / 全员装备养成。

---

## 一、定位分析

弭弗定位很明确：**围绕猛击 & 破防层数来打爆发的物理近卫**。

简单来说，他和管理员、大潘一样，都是吃队伍提前叠好的破防层数，然后通过猛击打出爆发伤害的角色。不同之处在于，弭弗不仅有猛击爆发，还自带一定程度的**聚怪、突进、生存护盾和霸体**。

### 手感 & 攻速

弭弗是大剑角色。在砍怪"卡肉"的情况下，约 **4.5 秒一轮重击**。

> **什么是卡肉**：攻击命中敌人时产生的受击停顿。比如双手剑对空气打完整套普攻可能只需 3.5 秒，但实际打到敌人时，每次命中都会插入一个很短停顿，累积起来约 4.5 秒。这个设计主要是增加武器的重量感和打击反馈。

虽然武器是大剑，但相比其他大剑角色，弭弗手感更流畅。加上**三段战技用拳头打人的机制**，操作感和打击感都非常强。

---

## 二、技能介绍

### 2.1 普攻：剑形拳意

四段物理伤害，一轮普攻卡肉情况下约 4.5 秒。弭弗虽是大剑角色，但手感不笨重，比以往大剑角色流畅很多。

---

### 2.2 战技（核心机制）

可以连续释放，一共分为三段，每段效果不同。

#### 一段：断云

- 拉取一定范围内敌人并造成伤害，**聚怪**。
- 面对拉不动的大型敌人时，不会把敌人拉过来，而是**自己反向突进到目标身前**。
- 打小怪聚怪，打大怪贴脸——清杂和追怪都非常灵活。
- 消耗 **100 技力**，命中后返还 **50 技力**。

#### 二段：追形

- 连续回旋造成伤害，最后一击带**猛击**效果。
- **重点**：如果二段消耗了**不少于 3 层破防**，就可以继续衔接三段战技「开天」。
- 消耗 **50 技力**。

#### 三段：开天

- 蓄力打出一拳，弭弗最核心的爆发点。
- 直线范围伤害（范围比看起来远，不是只能贴脸）。
- **不会继续消耗破防层数**，直接造成一次**视为猛击类型的高额伤害**。
- 专三倍率 **600%**，天赋还有 **1.2 倍率加成**——接近于多打了一个四层猛击的伤害。

> ⚠️ 第三段虽然是战技打出来的，但伤害类型是**猛击伤害**，吃的是**源石技艺强度**加成，不是普通战技伤害——吃不到装备上的战技伤害加成词条。

---

### 2.3 连携技：追形

发动后对目标造成物理伤害并施加**物理脆弱**，同时将战技替换为**二段战技**。

这个技能非常关键——不仅能给敌人挂上物理脆弱帮助触发天赋护盾，还能直接让弭弗进入二段战技，为后续衔接三段战技「开天」做准备，打出简易的爆发连招。

---

### 2.4 终结技：决心

短暂蓄力后向前冲刺，强制击飞前方敌人，并给命中敌人施加**1 层破防**，随后砸向地面造成物理伤害。

释放后战技直接变成二段，可以有更灵活的战斗策略。

**核心作用**：关键时刻补充破防层数——比如敌人当前破防层数不够，或队伍循环差一点破防时，用终结技补上这层破防，让后续猛击爆发打出更高伤害。

---

## 三、输出手法

### 基础连招

```
敌人 3 层破防 → 弭弗连携技 → 二段战技 → 三段战技
```

即：先把敌人破防层数叠好 → 触发弭弗连携技，挂物理脆弱 → 二段战技消耗破防层数打猛击 → 三段战技「开天」打出爆发。

# 【萌新推荐】弭弗x陈千语x埃特拉x阿列什 低配高伤&无脑循环打法教学

## 一、阵容概述

大家好，我是月咒，专注终末地战斗攻略的主播。

今天带来的这个阵容是给角色池子比较浅的玩家准备的。这套队伍的优势很明显：**续航能力优秀，低配也有不错的伤害，而且操作特别简单**。

它有一套非常稳定的**无限循环轴**，每一轮输出流程基本都一样，实战中稳定性极高，几乎不会因为 boss 转阶段、技能空掉等情况导致整套循环乱掉。

这套队伍属于**单核碎冰队**——虽然说是碎冰队，但大部分伤害其实都在弭弗身上。埃特拉和阿列什不需要太高的练度，整体成本很低。

## 二、排轴教学

> 以 0+1 弭弗的阵容为例。本视频主要讲热启动。

冷启动和热启动的区别：开局第一个技能不同。
- **冷启动**：用埃特拉战技给敌人挂一层寒冷附着
- **热启动**：用阿列什的大招给敌人挂一层寒冷附着

除了开局第一个技能不同，后续所有技能顺序和排轴**完全一样**。

---

### 2.1 第一波：爆发轴

1. **阿列什**的大招给 boss 挂一层**寒冷附着**。
2. **阿列什**的战技**爆掉附着**，给敌人挂**冰冻**。
3. 触发并使用**埃特拉的连携技**，给 boss 挂**第 1 层破防**。
4. **陈千语和阿列什的连携技**给 boss 挂**第 2 层破防**，同时给队友回技力。
5. **陈千语**的战技挂**第 3 层破防**。
6. **埃特拉**的大招挂**第 4 层破防**。
7. 触发并使用**弭弗的连携技**，给 boss 挂上物理脆弱，同时解锁二段战技。
8. **弭弗**的二段战技打**猛击**。
9. 陈千语的大招**不加入循环**，可以自由释放。
10. **弭弗**的三段战技打追加伤害收尾。

> 第一波爆发轴结束。

---

### 2.2 第二波（同上）

第二波打一套**一模一样的连招**。

> 因为这套打法本身破防层数已经够用了，所以弭弗大招和陈千语大招都不加入循环。陈千语大招可以等 boss 失衡时再放，用来打更高伤害；弭弗大招则可以根据情况使用——补失衡，或者某一波缺破防层数时拿来补层。

1. **阿列什**的大招挂寒冷附着。
2. **阿列什**的战技爆掉附着 → 冰冻。
3. **埃特拉**的连携技挂第 1 层破防。
4. **陈千语 + 阿列什**的连携技挂第 2 层破防，回技力。
5. **陈千语**的战技挂第 3 层破防。
6. **埃特拉**的大招挂第 4 层破防。
7. 触发**弭弗的连携技**挂物理脆弱，解锁二段战技。
8. **弭弗**二段战技猛击。
9. **弭弗**三段战技追加伤害收尾。

---

### 2.3 第三波

需要等阿列什大招转好才能启动（尽可能技力一格以上）。

1. **阿列什**大招挂寒冷附着。
2. **阿列什**战技触发三人连携。
3. **陈千语**战技 + **埃特拉**大招。
4. **弭弗**的连携技 → 二段战技 → 三段战技收尾。

### 3.4 阿列什

- **装备**：散件成套 + 充能手，堆充能。同时也要堆智力——用来提高触发**真灵**的概率（连携技有概率触发全队加攻 15% 的增益）。
- 这套打法里阿列什的充能需求非常高，即使是满潜也需要大概 **204.82%** 充能才能保证一轮一大。

| 武器 | 充能策略 |
|------|---------|

| 兼程主导者（待确认） | 自带充能词条，可以减少装备精炼次数 |

可以说这是目前终末地里少有的高强度懒人队：操作简单，爆发直接。

## 二、热启动排轴教学

> 热启动用满潜号演示，冷启动用 0+1 号演示。0+1 弭弗实测大概一分钟左右干掉 boss，伤害也非常高。不管什么潜能打法都一样，不用担心。
>
> 这个队伍谁主控都可以，但因为没有治疗，建议用弭弗。

### 2.1 第一波爆发

1. **弭弗**一段战技突进 → 二段战技，给 boss 挂**1 层破防**。
2. **汤汤**的战技给 boss 挂**1 层寒冷附着**。
3. 同时触发三个人的连携技——**陈千语、汤汤、洛茜**同时使用。
4. **洛茜**的二段连携技，此时 boss 身上已有**四层破防**。
5. **弭弗**的连携技给 boss 挂物理脆弱，同时解锁弭弗二段战技。
6. **弭弗**二段战技**猛击**直接爆掉四层破防。
7. **弭弗**三段战技追加伤害，完成这波爆发。

---

### 2.2 第二波：四大连开

第一波打完继续第二波，直接**四大连开**——大招顺序不影响，怎么开都行。

1. **弭弗**开大，给 boss 挂 1 层破防。
2. **汤汤**开大定 boss。
3. **洛茜**开大打输出，顺便挂上灼热附着。
4. **陈千语**开大给伤害和失衡。
5. 触发并使用**陈千语的连携技**，给 boss 挂 1 层破防。
6. **洛茜**连携一段 → 连携二段，给 boss 挂 2 层破防 → boss 四层破防。
7. **弭弗**连携技挂物理脆弱。
8. **弭弗**二段战技 → 三段战技完成斩杀。

## 三、冷启动排轴教学

冷启动和热启动打法基本一样。冷启动要打**三轮一模一样的连招**大招才能充好——不过这个轴在充能阶段伤害就非常高了，不是很依赖大招。只需要记住其中一段就学会了。

> 这组号是 0+1，大家可以感受一下伤害。

### 第一轮

1. 弭弗一段战技突进 → 二段战技猛击，挂 1 层破防。
2. 汤汤战技挂 1 层寒冷附着。
3. 同时触发陈千语 + 汤汤 + 洛茜的连携技，全部按掉。
4. 洛茜二段连携技 → boss 四层破防。
5. 弭弗连携技 → 弭弗二段战技猛击爆掉破防 → 三段战技结束。

> 打重击躲技能、恢复技力。下一轮需要**180 技力**（接近两格）才能启动。

### 第二轮（同上）

1. 弭弗一段战技突进 → 二段战技猛击，挂 1 层破防。
2. 汤汤战技挂 1 层寒冷附着。
3. 陈千语 + 汤汤 + 洛茜的连携技，全部按掉。
4. 洛茜二段连携技挂第四层破防。
5. 弭弗连携技挂物理脆弱 → 二段战技猛击 → 三段战技追加伤害。

### 第三轮（同上）

一模一样连招：弭弗战技一段突进二段猛击 → 挂破防 → 汤汤战技 → 三人连携 → 洛茜二段连携 → 弭弗连携 → 弭弗二段战技 → 三段战技。

然后**全员开大**。当充好的时候怪已经死了——如果还有血，四大连开就接到热启动教学的步骤继续循环。

# 【配队攻略】卡缪x洛茜x弭弗x骏卫 双先锋无限火力&装备养成推荐

>
> 阵容：卡缪 / 洛茜 / 弭弗 / 骏卫

---

## 一、阵容概述

大家好，我是月咒，专注终末地战斗攻略的主播。

今天给大家带来的是**卡缪 + 洛茜体系的双先锋阵容**。这套阵容是当前版本比较热门的配队之一，它最大的优点就是循环稳定、技力充足、伤害在线，而且操作反馈感很强。

不过这套队伍也有一个比较明显的缺点——不算特别轮椅。因为队伍里没有奶妈提供持续恢复，所以对操作和技能释放时机有一定要求。虽然这期视频里的排轴已经整得比较固定，实战流程也比较清晰，但相比一些带治疗、容错更高的阵容，这套双先锋确实更需要一点操作。

如果你喜欢主动操作，喜欢打连招和循环，这套阵容会非常适合你。

---

## 二、循环教学

冷启动的意思就是开局全员都没有大招。这套打法采用的是一波**充能轴** → 一波**爆发轴**的循环方式，打完一轮爆发后继续进入下一轮充能，形成无限循环。

> 本期演示双 C 使用的是一级武器，主要是为了展示这套阵容的循环逻辑。这个阵容谁主控都可以，选择自己顺手的来。

---

### 2.1 第一轮：冷启动·充能轴

1. **弭弗**起手，一段战技突进接二段战技，给 boss 挂 **1 层破防**。
2. **卡缪**的战技挂 **1 层灼热附着**。

> 需要记住：**洛茜的连携技**需要 1 层破防 + 1 层灼热附着才能触发。

3. **洛茜**第一段连携技，第二段在**金光**的时候使用，给 boss 叠至 **3 层破防**。
4. 3 层破防触发**弭弗的连携技**，洛茜刚才的连携技触发**卡缪的连携技**，直接使用。
5. **洛茜**的战技给 boss 挂**第 4 层破防**和**爪印斫痕** buff（洛茜游戏内本名为「洛茜」；爪印斫痕是其**天赋** debuff：物理伤害+12%、灼热伤害+12%，持续 25 秒）。
6. 4 层破防时，**弭弗**的二段战技猛击**爆掉破防**。
7. 顺手按**骏卫的连携技**恢复一点技力。
8. **弭弗**的三段战技「开天」追加伤害收尾。

> 充能轴结束。

---

### 2.2 第二轮：爆发轴

爆发轴和充能轴非常像。先用主控打重击恢复技力，等技力两格的时候再启动。

> **骏卫的大招好了就放，不需要加入循环，亮了就按就行。**

1. 和充能轴一样，**弭弗**一段战技突进接二段战技猛击，给 boss 挂 **1 层破防**。
2. **卡缪**的战技挂 **1 层灼热附着**。骏卫大招亮了直接开。
3. **洛茜**第一段连携技，在**金光**时用第二段，给 boss 挂至 **3 层破防**，触发并使用弭弗和卡缪的连携技。
4. **卡缪**每次使完连携技后，**血条前面会出现三把剑的图标**，这就是**连携 buff**，说明已经成功触发。看到这个图标后再去释放战技或终结技，下一个技能可以吃到连携的增强效果。所以尽可能等连携 buff 出来后再去打关键伤害技能。
5. **洛茜**的战技给 boss 挂**第 4 层破防**和**爪印斫痕** buff（洛茜游戏内本名为「洛茜」；爪印斫痕是其**天赋** debuff：物理伤害+12%、灼热伤害+12%，持续 25 秒）。
6. 开**卡缪的终结技**恢复技力——这个大招会送一个卡缪的**强化连携技**，需要通过卡缪的战技触发。
7. 用一次**卡缪的战技**（也就是强化连携技），触发连携。可以看到血条前面多了连携 buff。
8. **洛茜**的大招吃掉这个连携 buff 打爆发。
9. 过程中用**弭弗**的二段战技猛击爆掉 **4 层破防**，接三段战技「开天」追加伤害收尾。

> 爆发轴结束。

---

### 2.3 第三轮：充能轴

这一轮的技能顺序和第一轮基本一样，但有个小区别：

因为上一轮爆发时，卡缪和洛茜的大招都会给 boss 挂灼热附着，现在 boss 已经有**两层灼热附着**了——也就是说我们**少了一个挂灼热附着的过程**，相当于节省了一次技力。接下来只需要给 boss 挂一层破防，队伍就可以继续正常循环。

1. **弭弗**一段战技接二段，给 boss 挂 **1 层破防**。
2. **洛茜**一段连携技 + 二段连携技，给敌人叠至 **3 层破防**，触发并使用弭弗和卡缪的连携技。
3. **洛茜**的战技，给 boss 挂**第 4 层破防**。
4. **弭弗**二段战技猛击爆掉 4 层破防，接三段战技「开天」收尾。

> 充能轴结束。

---

### 2.4 第四轮：爆发轴

现在进入第四轮爆发轴。骏卫大招好了就开。整体打法和前面的爆发轴基本一样，但有个小区别：

此时弭弗的大招已经充能好了，可以用**弭弗的大招**给 boss 挂一层破防——和上一轮一样，相当于**又节省了 100 技力**，让后面的循环更加轻松。

1. **弭弗**大招破防挂完。
2. **卡缪**的战技给 boss 挂灼热附着。
3. **洛茜**一段连携技 + 二段连携技，给 boss 叠至 3 层破防，触发并使用弭弗和卡缪的连携技。
4. **洛茜**的战技挂至第 4 层破防。
5. **卡缪**开大招恢复技力，动画结束后再用卡缪的**强化连携技**（即他的战技）拿到连携 buff。
6. **洛茜**的大招打爆发。
7. 过程中用**弭弗**的二段战技接三段战技收尾。

> 整个过程中，骏卫的连携技都会在弭弗猛击爆掉破防之后触发，亮了按就可以了。

## 三、装备养成推荐

# 【配队攻略】莱万汀+卡缪 火队排轴教学 简易实战轴&进阶轴装备养成推荐

>
> 阵容：卡缪 / 莱万汀（42 / 史尔特尔） / 狼卫 / 艾尔黛拉（小羊）

---

## 一、阵容概述

大家好，我是月咒，专注终末地战斗攻略的主播。

这期来讲**莱万汀（42）+ 卡缪火队**的排轴攻略。内容分成三个部分：

1. **简易轴**：稳定性更高、操作更简单的打法。连续打两波一模一样的充能轴，给 boss 挂上艾尔黛拉的脆弱 buff，然后全员开大直接打爆发。
2. **进阶轴**：操作难度更高，但启动更快。核心是在第二波充能轴里使用狼卫的三火连招，节省 100 技力，让队伍更快进入爆发阶段。
3. **配装推荐**：这个打法对卡缪和狼卫的大招充能有一定要求。

> 两种打法各有优点——想打得稳一点用简易轴，想提速、操作感更强就用进阶轴。大家可以根据自己的习惯和练度自由选择。

---

## 二、冷启动·简易轴

冷启动就是全员开局没有大招。这种情况需要先打**两套一模一样的充能轴**，把 42 的大招先攒出来。这两套充能轴的技能顺序一样，记住一套就可以重复打。

> 在这个阵容里，主控的普攻不能停——要不断用重击去吸收 boss 身上的灼热附着，把它转化为 42 的熔火层数。所以实战中不是放完技能就站着等，而是技能之间要穿插重击，保持普攻不断。

---

### 2.1 第一波充能轴

1. **卡缪**的战技给 boss 挂**第 1 层灼热附着**。
2. 触发使用**狼卫的连携技**，挂**第 2 层灼热附着**。
3. 再次使用**卡缪的战技**，挂**第 3 层灼热附着**。

> 此时 boss 身上有 3 层灼热附着。42 的机制：主控打重击时可以把 boss 身上的灼热附着吸过来，转化为自身的**熔火**层数。如果开局三层没有一次性挂上也没关系——比如刚挂上两层就被重击吸走了，分几次吸都可以。

4. 主控打**重击**，把 boss 身上的火吸过来。
5. 触发**艾尔黛拉和卡缪的连携技**，直接使用。艾尔黛拉的连携会给 boss 挂一个腐蚀 buff；卡缪的连携会提供一层**连携 buff**。
6. 使用**42 的连携技**获得一层熔火（42 的连携本身就会增加一层）。

> 此时 42 血条前应出现**四个红色花瓣**，代表**四层熔火**。

7. 满熔火时，用**42 的强化战技**完成第一波充能。

---

### 2.2 第二波充能轴

第一波打完还需要再打一套。但此时技力不太够，大家的连携也都在冷却——不要着急开下一轮。

1. 先用**普攻接重击**继续恢复技力，顺便等连携技转好。
2. 等技力回到两格时再启动下一波充能轴。
3. **卡缪**的战技挂第 1 层火 → **狼卫**的连携技挂第 2 层火 → **卡缪**的战技挂第 3 层火。
4. 主控打**重击**，把三层灼热附着吸过来。
5. 使用**42 和卡缪的连携技**（42 的连携由艾尔黛拉的连携触发）。使用后 42 的熔火增加到 4 层（四个花瓣又红了）。
6. **42 的强化战技**完成充能。

---

### 2.3 爆发阶段

1. 如果打的是清杂环境，42 可以直接开大爆发。但对单打 boss 时需要等**艾尔黛拉的战技**。
2. 重击触发艾尔黛拉的战技挂一个腐蚀 buff（腐蚀延迟生效）。看到爆炸特效 / boss 身上有腐蚀 buff 时，再用艾尔黛拉的战技挂**脆弱** buff（还能触发专武异常效果，增加拐力）。
3. 再用一次**42 的连携技**（42 的连携可以通过闪避取消后摇，亮了就按即可）。
4. **42 开大**准备爆发。
5. **狼卫**开大打失衡。艾尔黛拉开不开大都行——开大可以帮忙打伤害，留着当加血金刚也不错。

#### 开大后的操作

- 按住普攻键，连携亮了就按，四层熔火就打**强化战技**。
- 普攻第三刀时会给 boss 挂灼热附着并触发狼卫的连携。
- 打重击吸火，触发卡缪的连携。

> **关键节奏**：卡缪的连携给连携 buff（血条前出现小剑图标）。看到图标说明连携 buff 拿到了，此时在四层熔火状态下打 42 的强化战技——这招吃满 buff，伤害非常高。

- 打 42 的强化战技（打伤害 + 挂燃烧），再用 42 的连携技，重击保持不断。
- 开**卡缪的大招**准备拿第二个连携 buff。
- 重击触发艾尔黛拉的连携，直接使用。
- 卡缪开大后，等卡缪的战技亮了就按——这是大招送的**强化连携技**，通过战技触发。
- 继续重击不断，吸收到四层熔火 + 连携 buff，再打一次 42 的强化战技（又是吃满伤害的）。

> 42 大招快结束时，如果场上还有敌人，可以攒一波四层熔火，等大招结束后补一个强化战技。这样 42 的大招充能直接回到一半左右，后面只需再打一轮充能轴就可以继续开大爆发。

## 三、冷启动·进阶轴

进阶轴也分两波充能轴。**第一波和简易轴完全一样**，第二波有区别。

### 3.1 第一波（同简易轴）

普攻重击不断 → 卡缪战技挂第 1 层灼热附着 → 狼卫连携挂第 2 层 → 卡缪战技挂第 3 层 → 重击吸火 → 触发艾尔黛拉和卡缪的连携技 → 42 连携技（四层熔火） → 42 强化战技完成第一波充能。

### 3.2 第二波（进阶）

第一套打完，主控打重击恢复技力，等连携冷却转好，技力两格时启动。

1. 先触发**艾尔黛拉的连携**。艾尔黛拉连携腐蚀挂上后，再用艾尔黛拉的战技给 boss 挂**脆弱 + 专武异常** buff。
2. 用**42 的连携**获得一层熔火。

> **这里是进阶轴的重点**：

3. **卡缪**的战技给 boss 挂第 1 层灼热附着。
4. 使用**狼卫的连携技**——注意看狼卫的动作，在他**后空翻即将落地但没有落地时**，用**狼卫的大招取消连携后摇**。火已经挂上了，大招结束后还会再次触发狼卫连携。
5. 重击吸的火用**卡缪的连携**处理。

> 这个打法需要狼卫的**五潜**。

6. 把三层火用重击**全部转化**为 42 的熔火。
7. 满熔火后打**42 的强化战技**完成此次充能。

后续大招好了以后，和简易轴的爆发阶段一样，不再重复演示。

## 四、装备养成推荐

### 4.1 42（史尔特尔）

| 方案 | 配置 | 说明 |
|------|------|------|
| **极限伤害套** | 零潜要打 3 个连携技，满潜要打 2 个连携才能充满大招，只为伤害拉满 | 觉得 42 伤害不够炸时非常推荐 |
| **新手充能套** | 动火用 + 充能衣服（两件衣服二选一，一个攻击高一点、一个血量高一点，区别不大） | 更简单，推荐新手穿这套 |

---

### 4.2 卡缪

**充能阈值（最低要求：武器 + 装备数值）**：

| 潜能 | 基础阈值 | 属性面板显示 | 42 攻击 > 1000 时 |
|------|----------|-------------|-------------------|
| 零潜 | 58.54% | 158.54% | 66.45% |
| 满潜 | 34.76% | — | 41.49% |

> 42 攻击在 1000 以上时，返还技力会让队友的充能阈值提高。以上是最低充能要求，尽可能堆高一些。

---

### 4.3 狼卫

| 方案 | 配置 | 说明 |
|------|------|------|
| **减 CD 套** | 3 件清波 + 长息披帘 | 减少连携冷却。进阶轴穿这套也可以，但要把武器 O.B.J.迅极的充能词条升满，装备充能词条也要精炼。如果只打简易轴，不太需要充能，无需精炼。 |
| **充能套（进阶轴）** | 4 件充能 + 3 件长息 | 多堆力量词条，套装加一千血。有其他精炼过的充能装备也可以穿别的。目标：充能堆到 212.5%。 |

---

### 4.4 艾尔黛拉（小羊）

- **装备**：长息套装（三件套），既能保证充能也能保证奶量。
- **技能加点**：只加战技即可。
