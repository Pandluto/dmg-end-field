# 开发修改 0.2.0 - 代码重构与三栏布局

## 约束
- 本计划仅针对代码重构和三栏布局升级
- 不修改业务逻辑（拖拽、吸附、删除等功能保持不变）
- 不修改样式视觉设计
- 仅调整代码结构和布局配置

## 背景

当前 `CanvasBoard/index.tsx` 存在以下问题：
1. ** monolithic 组件** - 436行代码全部集中在一个组件
2. **职责不清** - 渲染逻辑、拖拽逻辑、布局计算混杂
3. **布局限制** - 现有左右结构无法灵活扩展功能

## 问题分析

### 问题1：代码架构过于集中

**当前结构：**
```
CanvasBoard/index.tsx (436行)
├── 状态管理 (useState, useRef)
├── 拖拽逻辑 (useEffect, handlers)
├── 布局计算 (findNearestLine, snapToNearestNode)
├── 渲染函数 (renderStaffGroup, renderLines, renderSkillButtons, renderDraggingButton)
└── JSX (全部混合在一起)
```

**期望结构：**
```
CanvasBoard/
├── index.tsx              # 主容器，负责布局组合
├── CanvasArea.tsx         # 画布区域（含谱线和节点）
├── Toolbar.tsx            # 工具栏
├── SidePanel.tsx          # 左侧面板（新增）
├── DraggingOverlay.tsx     # 拖拽悬浮层
├── hooks/
│   ├── useCanvasDrag.ts   # 拖拽逻辑 hook
│   ├── useCanvasWidth.ts  # 宽度响应式 hook
│   └── useSelectStart.ts  # 选择文本阻止 hook
├── utils/
│   └── canvasRenderer.ts  # 谱线/节点渲染工具函数
└── types.ts               # CanvasBoard 私有类型
```

### 问题2：布局结构限制功能扩展

**当前布局（左右结构）：**
```
.canvas-layout (display: flex)
├── .canvas-area (flex: 1)
└── .skill-sandbox
```

**目标布局（三栏结构）：**
```
.canvas-layout (display: flex)
├── .side-panel (width: 25%, min-width: 200px)
├── .canvas-area (flex: 1)
└── .skill-sandbox
```

类比 VSCode IDE 布局：
- 左侧 = 资源管理器/侧边栏
- 中间 = 编辑器区域
- 右侧 = 技能沙盒（可类比属性面板）

## 技术方案

### 1. 代码解耦方案

#### 1.1 提取 CanvasArea 组件

将谱线和节点的渲染逻辑提取为独立组件：

```typescript
// CanvasArea.tsx
interface CanvasAreaProps {
  config: CanvasConfig;
  staffCount: number;
  selectedCharacters: Character[];
  skillButtons: SkillButton[];
  onButtonMouseDown: (e: React.MouseEvent, buttonId: string) => void;
  onButtonContextMenu: (e: React.MouseEvent, buttonId: string) => void;
  onCanvasClick: () => void;
}

export function CanvasArea({ ... }: CanvasAreaProps) {
  // 纯粹的渲染逻辑，无状态
  return (
    <div className="canvas-area">
      <Toolbar />
      <CanvasContainer>
        {renderLines()}
        {renderSkillButtons()}
      </CanvasContainer>
    </div>
  );
}
```

#### 1.2 提取 DraggingOverlay 组件

拖拽时的悬浮按钮独立渲染：

```typescript
// DraggingOverlay.tsx
interface DraggingOverlayProps {
  draggingState: DraggingState | null;
  mousePosition: { x: number; y: number };
  buttonSize: number;
}

export function DraggingOverlay({ ... }: DraggingOverlayProps) {
  if (!draggingState) return null;
  return (
    <div className="skill-button dragging" style={...}>
      {draggingState.skillType}
    </div>
  );
}
```

#### 1.3 提取 SidePanel 组件

新增左侧面板：

```typescript
// SidePanel.tsx
interface SidePanelProps {
  widthPercent: number;  // 默认 25
}

export function SidePanel({ widthPercent }: SidePanelProps) {
  return (
    <div className="side-panel" style={{ width: `${widthPercent}%` }}>
      <div className="side-panel-content">
        {/* 预留扩展区域 */}
        {/* 可配置功能：时间轴、快捷键配置、角色详情等 */}
      </div>
    </div>
  );
}
```
