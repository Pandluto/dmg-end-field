# 开发修改 0.2.1 - 0.2.0 重构 Bug 修复

## 约束
- 本计划仅针对 0.2.0 重构后引入的 bug 修复
- 不修改任何重构结构
- 仅修复以下三个具体问题

## 问题分析

### Bug 1：技能按钮节点吸附功能失效

**表现：**
拖拽技能按钮到谱线时，按钮没有吸附到最近的节点，而是停留在拖拽释放的位置。

**根因分析：**
`useCanvasDrag` hook 中 `handleMouseUp` 逻辑存在问题：
- `snapToNearestNode` 函数调用时传入的 `skillButtons` 参数可能为空或未正确更新
- `findNearestLine` 返回的 `lineY` 可能未正确加上 `groupOffset`

**检查位置：**
- `hooks/useCanvasDrag.ts` 第 100-120 行

### Bug 2：预留扩展区域 CSS 错误

**表现：**
左侧面板背景色或样式显示异常，未保持白底。

**根因分析：**
`CanvasBoard.css` 中 `.side-panel` 使用了 `--bg-secondary` 变量，但该变量可能未定义或与预期不符。

### Bug 3：预留扩展区域宽度过大

**表现：**
左侧面板宽度为 25%，占用空间过大。

**根因分析：**
`SidePanel` 组件的默认 `widthPercent` 为 25。

### Bug 1 修复

检查 `useCanvasDrag.ts` 中 `handleMouseUp` 的 `snapToNearestNode` 调用：

```typescript

  draggingState.characterId

```

### Bug 2 修复

修改 `.side-panel` 背景色：

```css
.side-panel {
  background: #ffffff;  /* 白底 */
}
```

### Bug 3 修复

修改 `SidePanel` 默认宽度：

```typescript
export function SidePanel({ widthPercent = 20 }: SidePanelProps) {
```
