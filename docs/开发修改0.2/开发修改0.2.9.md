# 开发修改 0.2.9 - 技能沙盒按钮添加技能类型文字标注

## 约束

- 本计划仅为技能沙盒的技能按钮添加右侧技能类型标注
- 标注内容为：重击（A）、战技（B）、连携（E）、终结（Q）
- 不修改拖拽、吸附、渲染等核心逻辑
- 资源路径严格遵循规范，**禁止臆造**

---

## 需求分析

### 当前状态

技能沙盒（`SkillSandbox`）右侧面板中，每个技能按钮仅显示：
- 技能图标（优先）或
- 技能类型字母（A/B/E/Q）

### 目标状态

在不操作技能按钮时（鼠标未 hover/click），按钮右侧显示技能类型中文标注：

```
[●] 重击    [●] 战技
[●] 连携    [●] 终结
```

其中 `[●]` 为技能按钮（圆形），右侧文字为技能类型标注。

### 标注映射

| 技能类型 | 显示标注 |
|---------|---------|
| A | 重击 |
| B | 战技 |
| E | 连携 |
| Q | 终结 |

---

## 技术方案

### 方案选择

**采用绝对定位文字标注层**（在 `.sandbox-skills` 外层添加标签容器），原因：
1. 技能按钮已有固定尺寸（44×44px），标签位置可精确计算
2. 不影响现有按钮拖拽功能（仅视觉标注）
3. 易于批量渲染，与按钮一一对应

### 实现方式

在 `SkillSandbox.tsx` 中：
1. 定义技能显示标签常量 `SKILL_DISPLAY_LABELS`
2. 调整 `.sandbox-skills` 容器为**相对定位**作为参照
3. 每个技能按钮的**右侧**绝对定位一个 `<span>` 标签显示中文名称
4. 标签样式：`position: absolute; left: 48px; top: 50%; transform: translateY(-50%)`

### CSS 调整

修改 `SkillSandbox.css`：
1. `.sandbox-skills`：设置 `position: relative` 作为参照
2. 新增 `.sandbox-skill-label` 样式：绝对定位右侧，显示中文技能名

## 实现步骤

### 步骤 1：修改 `src/components/CanvasBoard/SkillSandbox.tsx`

在组件中：
1. 引入 `SKILL_LABELS`（已引入）
2. 添加技能显示标签常量：

```typescript
/** 技能显示标签（用于按钮右侧标注） */
const SKILL_DISPLAY_LABELS: Record<SkillType, string> = {
  A: '重击',
  B: '战技',
  E: '连携',
  Q: '终结',
};
```

3. 调整 `.sandbox-skills` 网格容器结构（每个技能按钮后跟一个标签 span）

### 步骤 2：修改 `src/components/CanvasBoard/SkillSandbox.css`

1. 将 `.sandbox-skills` 设置为 `position: relative`
2. 新增标签样式：

```css
.sandbox-skill-tag {
  position: absolute;
  left: 48px;
  top: 50%;
  transform: translateY(-50%);
  font-size: 12px;
  color: #666;
  white-space: nowrap;
  pointer-events: none;
}
```

### 步骤 3：验证

1. 启动项目 `npm run dev`
2. 打开画布界面，观察技能沙盒右侧是否显示"重击/战技/连携/终结"标签
3. 拖拽技能按钮，确认标签不干扰拖拽功能
