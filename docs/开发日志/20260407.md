## 已实现功能

### 选人界面
- 1-4角色选择
- 角色数据显示（名称、属性、武器等）
- 复选框选择

### 画布排轴界面
- 横向多行谱线显示
- 2-5组谱线（可动态调整），每组4条（共8-20条）
- 节点渲染（40节点/线，22px间隔）
- 技能沙盒：显示选中角色的A/B/E/Q技能按钮
- 拖拽功能：
  - 从沙盒拖出技能按钮到谱线
  - 按钮跟随鼠标移动（整屏范围）
  - 释放时吸附到最近的节点
  - 角色匹配检查（管理员按钮只能吸附到管理员谱线）
- 已放置按钮的拖拽移动
- 右键删除按钮
- 防止右键长按时选中文本（mousedown preventDefault + selectstart 全局捕获）
- 谱线右侧长度缩短（right: 20px）

## 配置参数
```typescript
canvasWidthPercent: 0.6    // 画布宽度60%
canvasHeight: 600
staffCount: 2              // 2-5组谱线
lineCount: 4               // 每组4条线
nodeCount: 40              // 每线40节点
nodeSpacing: 22             // 节点间隔22px
skillButtonSize: 44        // 按钮直径44px
groupSpacing: 0            // 组间间距（已缩小）
```

## 核心算法

### findNearestLine
根据鼠标Y坐标，找到最近的匹配角色谱线
- 遍历所有谱线
- 过滤保留角色ID匹配的谱线
- 返回最近的谱线信息

### snapToNearestNode
将位置吸附到最近的节点
- 基于 nodeSpacing 计算
- 四舍五入找到最近节点
- 防重叠检查
