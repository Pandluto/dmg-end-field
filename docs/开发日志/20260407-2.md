## 开发修改记录

### 0.2.4 - 技能沙盒添加头像和技能图标

**目标**：为技能沙盒的干员标识和四个技能按钮分别添加图片

**实现**：
- 头像路径：`/assets/avatars/<name>/<name>.png`
- 技能图标路径：`/assets/avatars/<name>/<name><skillName>.png`
  - A → 普通攻击
  - B → 战技
  - E → 连携技
  - Q → 终结技

**修改文件**：
- `src/types/index.ts`：Character 接口新增 `avatarUrl`、`skillIconMap` 字段
- `src/context/AppContext.tsx`：注入头像和技能图标 URL
- `src/utils/assetResolver.ts`：新增 `resolveAvatarUrl`、`resolveSkillIconUrl` 函数

### 0.2.8 - 修复连续吸附时 y 轴偏移 Bug

**问题**：连续拖拽吸附多个按钮时，y 轴在 62（正确）和 362（错误）之间跳变

**根因**：`snapToNearestNode` 重复计算 `groupOffset`

**修复**：
- `findNearestLine` 返回的 `lineY` 已含 `groupOffset`
- `snapToNearestNode` 直接使用 `position.y`（不重复加 groupOffset）

**修改文件**：`src/utils/layout.ts`

---

## Bug 修复

1. **y 轴双重偏移**：移除 `snapToNearestNode` 中重复的 `groupOffset` 计算
2. **按钮堆叠**：将 `position: relative` 改为 `position: absolute` 使 `left/top` 生效
3. **SKILL_COLORS 未使用**：确认删除后无引用，已清理
4. **图标消失**：`SkillButton` 接口补充 `skillIconUrl` 和 `element` 字段
