# 打包版快速上手

## 入口

双击 exe。

- `打开浏览器 Web`：进入主界面。
- Shell 左侧 `数据`：应用/保存数据。
- Shell 左侧 `图片`：下载/切换图片。

## 应用数据

Shell 点 `数据`。

1. 点一个存档。
2. 勾选要应用的内容。
3. 点 `应用这个存档`。
4. 回 Web 刷新。

要备份当前数据：填名字，点 `保存到本机` 或 `保存到共享`。

## 下载图片

Shell 点 `图片`。

1. 点 `检查更新`。
2. 点 `下载并切换`。
3. 回 Web 刷新。

零散图片：Web 底部 `图片管理` -> `导入`。

## 主界面

1. 先选干员，最多 4 个。
2. 点 `开始排轴`。
3. 没有干员就先点 `编辑干员`。

## 干员配置

入口：底部 `角色配置`，或顶部抽屉 `干员配置`。

顺序：

1. 右侧头像切干员。
2. 调等级、潜能、主能力、副能力。
3. 选武器，调武器等级、潜能、效果等级。
4. 选装备，调词条等级。
5. 调 A/B/E/Q 技能等级。

## 排轴

- 右边技能拖到左边格子。
- 拖节点改位置。
- 点节点看伤害详情。
- 右键节点删除、复制、换技能。
- 底部 `保存` / `恢复` / `分享` 管排轴。

## 上 Buff

点技能节点进详情。

1. 右上角点 `+`。
2. 搜干员、武器、装备、通用 Buff 或异常状态 Buff。
3. 选中后加入。
4. 在 Buff 列表里删、禁用、启用、加减层数。

## 微调

技能详情里：

- `Hit 微调`：单段开关 Buff、调层数。
- 准星按钮：改目标抗性。
- 锁按钮：防止节点误拖。
- 右侧计算过程：看 Buff 进了哪个乘区。

## 批量 Buff

顶部抽屉点 `BUFF批量操作`。

1. 点选或框选技能节点。
2. 点 `+` 批量增加，点 `-` 批量删除。
3. 选 Buff。
4. 点目标节点。
5. 点确认。

漏斗：按已有 Buff 筛节点。  
编辑目录：批量补齐或移除共同 Buff。

## 自定义数据

底部入口：

- `编辑干员`
- `编辑BUFF`
- `编辑武器`
- `编辑装备`
- `图片管理`

改完后回主界面刷新或重新选干员。

## AI CLI

入口：底部 `AI CLI`。

常用：

```text
help
fill.task
fill.check <JSON>
fill.apply <JSON>
proposal.approve #1
proposal.save #1
```

流程：`*.fill.task` 给 AI -> AI 回 JSON -> `*.fill.check` -> `*.fill.apply` -> 按 `Y` -> 再按 `Y`。

`apply` 只是提案，必须批准并保存。
