# 核心概念

## 属性系统

### 四维普通属性

| 中文 | 英文 | 主属性加成 | 副属性加成 |
|------|------|-----------|-----------|
| 力量 | strength | 0.5%/点 | 0.2%/点 |
| 敏捷 | agility | 0.5%/点 | 0.2%/点 |
| 智识 | intelligence | 0.5%/点 | 0.2%/点 |
| 意志 | will | 0.5%/点 | 0.2%/点 |

### 能力值加成公式

```
能力值加成 = 主能力 × 0.005 + 副能力 × 0.002
```

### 角色面板构成

1. **角色基础数据** - 等级属性、好感加成
2. **武器数据** - 基础攻击、阶段属性、珠子加成
3. **装备数据** - 固定值、百分比词条

### 攻击力计算公式

```
攻击力 = [(角色基础攻击力 + 武器基础攻击力) × (1 + 百分比加成) + 固定数值加成] × (1 + 能力值加成)
```

**默认值：**
- 角色默认 90 级
- 武器默认 90 级
- 好感额外提供 +60 主能力
- 暴击率默认 5%，暴击伤害默认 50%

## 暴击期望倍率

```
暴击期望倍率 = 1 + min(暴击率, 1) × 暴击伤害
```

## 角色配置黑话

| 格式 | 含义 |
|------|------|
| `0+1某角色` | 角色零潜 + 专武0潜 |
| `0+0某角色` | 角色零潜 + 无专武 |
| `6+1某角色` | 角色满潜 + 专武0潜 |
| `6+5某角色` | 角色满潜 + 专武满潜 |

**规则：**
- 六星角色默认零潜
- 五星角色默认满潜（5潜）
- 专武默认0潜满珠子
- **管理员默认视为满潜**
