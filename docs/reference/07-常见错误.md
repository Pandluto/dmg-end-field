# 常见错误记录

## 错误 #1：装备加成理解错误

**优先级**: 🔴 高

**错误**: 错误地将装备加成理解为"含武器总加成"，导致重复扣除。

```
错误: equipment.mainStat = 279 - 156(武器) = 123 ❌
正确: equipment.mainStat = 279（其他装备）✓
```

**预防**: `equipment` 参数不含武器，武器由计算器单独处理。

---

## 错误 #2：使用错误的 CLI 工具

**优先级**: 🟡 中

**错误**: 使用 `attack_cli.ts` 而非 `panel_cli.ts`。

- `attack_cli.ts` - 不支持装备加成
- `panel_cli.ts` - 支持完整面板计算

**预防**: 面板计算使用 `npm run panel`。

---

## 错误 #3：伤害加成命中判断错误

**优先级**: 🔴 高

**错误**: 按"技能类别是否匹配"判断，而非"加成能否作用于本次伤害"。

```
加成: 寒冷伤害+16%（某技能触发）
本次: 战技造成寒冷伤害

错误: 因为是战技所以不命中 ❌
正确: 寒冷伤害可直接作用，判定命中 ✓
```

**原则**:

```
是否命中 = 条件触发状态 × 加成可作用性
```

---

## 错误 #4：百分比直接当整数

**优先级**: 🔴 高

**错误**: `0.538` 当成 `53.8` 参与计算。

```
错误: ATK × 53.8% = ATK × 53.8 ❌
正确: ATK × 0.538 = ATK × 0.538 ✓
```

---

## 错误 #5：技能等级取错

**优先级**: 🔴 高

**错误**: 默认技能等级应为 9，专3才用 M3。

```
错误: 默认取 M3 ❌
正确: 默认取 9 ✓
```

---

## 错误 #6：多段伤害只算首段

**优先级**: 🔴 高

**错误**: 多段技能只计算第一段伤害。

```
正确: 总伤 = Σ(每段伤害)
```

---

## 错误 #7：潜能效果未区分

**优先级**: 🟡 中

**错误**: 条件生效的潜能写入 `stats` 字段。

```
# 正确区分
无条件生效: { "stats": { "agility": 15 } }
条件生效:   { "description": "..." }  // 无 stats
```

---

## 错误 #8：第三技能无条件/条件效果混淆

**优先级**: 🟡 中

**错误**: 把条件效果当作无条件计入面板。

```
# 正确结构
{
  "passive": { "atkPercent": 0.07 },  // 无条件 → 计入面板
  "description": "..."                 // 条件 → 不计入面板
}
```
