# 代码架构

## 核心模块

### 1. calculator.ts

**职责**: 核心伤害计算引擎

**主要函数**:

| 函数 | 功能 |
|------|------|
| `calculateFinalAtk()` | 计算最终攻击力 |
| `calculateCritMultiplier()` | 计算暴击期望 |
| `calculateDamage()` | 计算技能伤害 |
| `calculateAbnormalDamage()` | 计算异常伤害 |
| `calculateTeamDPS()` | 计算队伍 DPS |

**常量数据**:

```typescript
ENEMY_RESISTANCE      // 敌人抗性等级
CORROSION_DATA        // 腐蚀数值表
VULNERABILITY_DATA    // 导电/碎甲易伤表
ABNORMAL_MULTIPLIER    // 异常伤害倍率
ABNORMAL_DURATION     // 异常持续时间
```

---

### 2. panel_template.ts

**职责**: 角色面板计算

**主要函数**:

```typescript
calculatePanel(input: PanelTemplateInput): PanelTemplateResult
formatPanelResult(result: PanelTemplateResult): string
```

**输入参数**:

```typescript
interface PanelTemplateInput {
  characterName: string;
  characterLevel: number;
  weaponName?: string;
  weaponLevel?: number;
  matrixCode?: string;
  potential?: number;
  equipment?: Partial<EquipmentBonus>;
}
```

**计算流程**:

1. 加载角色 JSON → 获取等级属性
2. 加载武器 JSON → 获取基础攻击和技能加成
3. 解析珠子编码 → 计算技能等级
4. 叠加装备加成
5. 计算潜能加成
6. 汇总能力值和攻击力

---

### 3. damage_template.ts

**职责**: 技能伤害计算

**主要函数**:

```typescript
calculateSkillDamage(input: DamageCalculationInput): DamageCalculationResult
formatDamageResult(result: DamageCalculationResult): string
buildDamagePanelFromPanelResult(): DamagePanelInput
```

**输入参数**:

```typescript
interface DamageCalculationInput {
  characterName: string;
  skillType: 'normalAttack' | 'skill' | 'chainSkill' | 'ultimate';
  skillLevel?: '1'-'9' | 'M1'-'M3';
  panel: DamagePanelInput;
  damageBonus?: Partial<DamageBonusInput>;
  triggerConditions?: Record<string, boolean>;
  enemy?: Partial<EnemySideInput>;
}
```

---

## 数据流

```
用户输入
    ↓
panel_template.ts → calculatePanel() → PanelTemplateResult
    ↓
buildDamagePanelFromPanelResult() → DamagePanelInput
    ↓
damage_template.ts → calculateSkillDamage() → DamageCalculationResult
    ↓
formatDamageResult() → 格式化字符串输出
```

## 乘区实现

### 伤害加成区计算

```typescript
const zoneDamageBonus =
  1 +
  elementBonus +                    // 元素伤害加成
  skillCategoryBonus +              // 技能类型加成
  panel.allSkillDmgBonus +          // 所有伤害加成
  damageBonus.bonusUnconditional +  // 无条件加成
  bonusConditional +                // 有条件加成(已触发)
  damageBonus.bonusFromOthers;      // 他人加成
```

### 敌方侧计算

```typescript
const zoneFragile = 1 + enemySide.fragile;
const zoneVulnerability = 1 + enemySide.vulnerability;
const zoneDefense = enemySide.defenseZone;  // 默认 0.5
const zoneImbalanceVulnerability = 1 + enemySide.imbalanceVulnerability;
const zoneResistance = enemySide.resistanceZone;
```

## 异常系统

### 腐蚀 (Corrosion)

```typescript
calculateCorrosionResistReduction(corrosion, memoryStrength)
// 减抗(t) = min(初始减抗 + t × 每秒减抗, 最大减抗)
// 系数 = 1 + calculateMemoryCoefficient(ms)
```

### 导电/碎甲 (Conductivity/Shatter)

```typescript
calculateVulnerabilityData(level, memoryStrength)
// 易伤值 × (1 + 系数)
```
