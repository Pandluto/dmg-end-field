# 面板计算体系

## 基础攻击力计算

```
攻击力 = [(角色基础攻击 + 武器基础攻击) × (1 + 百分比加成) + 固定攻击] × (1 + 能力值加成)
```

## 能力值加成

```
能力值加成 = 主能力 × 0.5% + 副能力 × 0.2%
```

### 主能力系数加成

如果有 `mainAbilityPercent` 加成：

```
主能力' = 主能力 × (1 + mainAbilityPercent)
能力值加成 = 主能力' × 0.5% + 副能力 × 0.2%
```

---

## 面板计算流程

1. 加载角色数据 (等级属性)
2. 加载武器数据 (基础攻击、技能加成)
3. 计算珠子加成 (编码如663)
4. 计算装备加成
5. 计算潜能加成
6. 汇总所有加成

---

## 珠子系统

### 编码规则

珠子编码为三位数字，分别对应三个武器技能等级：

| 编码 | 含义 |
|------|------|
| 111 | 技能1级/技能2级/技能3级 |
| 663 | 技能16级/技能26级/技能33级 |

### 等级计算

```
武器技能最终等级 = 武器技能基础等级 + 珠子技能等级
最终等级 ≤ 阶段上限等级
```

### 默认值

- 起始编码: 111
- 满级编码: 663
- 未提及时默认: 663

## 武器技能类型

| 类型 | 说明 |
|------|------|
| agility | 敏捷提升 |
| strength | 力量提升 |
| intelligence | 智识提升 |
| will | 意志提升 |
| atkPercent | 攻击力百分比 |
| memoryStrength | 源石技艺强度 |
| critRate | 暴击率 |
| special | 特殊效果 |

---

## 第三技能特殊规则

### 无条件生效 (passive)

直接加到面板，参与计算：
- 攻击力+7%
- 源石技艺强度+30

### 条件生效

需要触发条件，**不参与面板计算**：
- 施加破防时终结技伤害+33.6%

---

## 潜能效果区分

### 无条件生效

有 `stats` 字段，直接加入面板：
```json
{
  "id": 2,
  "name": "家传武学",
  "stats": {
    "agility": 15,
    "physicalDmgBonus": 0.08
  }
}
```

### 条件生效

无 `stats` 字段，仅描述不参与计算：
```json
{
  "id": 1,
  "name": "绝影",
  "description": "对生命值少于50%的敌人伤害+20%"
}
```

## 装备加成参数

### 基础属性

| 字段 | 类型 | 说明 |
|------|------|------|
| mainStat | number | 主能力加成 |
| subStat | number | 副能力加成 |
| flatAtk | number | 固定攻击力 |
| atkPercent | number | 攻击力百分比 |

### 伤害加成

| 字段 | 说明 |
|------|------|
| physicalDmgBonus | 物理伤害加成 |
| fireDmgBonus | 灼热伤害加成 |
| electricDmgBonus | 电磁伤害加成 |
| iceDmgBonus | 寒冷伤害加成 |
| etherDmgBonus | 自然伤害加成 |
| skillDmgBonus | 战技伤害加成 |
| chainSkillDmgBonus | 连携技伤害加成 |
| ultimateDmgBonus | 终结技伤害加成 |
| allSkillDmgBonus | 所有伤害加成 |

### 暴击属性

| 字段 | 说明 |
|------|------|
| critRate | 暴击率 |
| critDmg | 暴击伤害 |

## 重要说明

### 装备加成不含武器

`equipment` 参数是"其他装备加成"，**不包含武器贡献**。

```
正确: equipment: { mainStat: 279 }  // 敏捷来自其他装备
错误: equipment: { mainStat: 123 }  // 错误地扣除了武器贡献
```

武器贡献由计算器根据 weaponName + matrixCode 自动计算。
