## 技能倍率结构

### 单段伤害

```json
"9": { "damage": 2.80 }
```

### 多段伤害

```json
"9": {
  "hit1": 0.41,
  "hit2": 0.49,
  "hit3": 0.54,
  "hit4": 0.62,
  "hit5": 0.72
}
```

### 特殊倍率键

- `execute` - 处决伤害
- `plunge` - 下落攻击
- `extraDamage` - 额外伤害
- `crystalDamage` - 结晶伤害

---

## 数值格式规范

- 百分比 → 小数: `23%` → `0.23`
- 等级 key: `"1"` 到 `"9"`, `"M1"`, `"M2"`, `"M3"`
- 阶段 key: `"0"` 到 `"4"`
- 潜能 id: `1` 到 `5`
