# EndSkill 项目概述

## 项目定位

EndSkill 是一个 **终末地 DPS 计算器** 项目，用于计算游戏中角色的面板属性和技能伤害。

## 核心技术栈

- **TypeScript** - 主要开发语言
- **Node.js** - 运行时环境
- **Vite** - 构建工具（项目配置）

## 项目结构

```
EndSkill/
├── src/                    # 源代码
│   ├── calculator.ts       # 核心伤害计算逻辑
│   ├── panel_template.ts   # 面板计算模板
│   ├── damage_template.ts  # 伤害计算模板
│   └── *.ts               # CLI 入口
├── data/                   # 游戏数据
│   ├── characters/         # 角色数据 (JSON + MD)
│   └── weapons/           # 武器数据 (JSON + MD)
├── Rules/                  # 规则文档
│   ├── 基础知识.md        # 核心数值规则
│   ├── 伤害计算.md        # 伤害公式规范
│   ├── 面板计算.md        # 面板计算规范
│   └── characters.md      # 角色数据说明
└── dist/                   # 编译输出
```

## 核心功能

| 模块 | 功能 | 入口文件 |
|------|------|----------|
| 面板计算 | 计算角色最终属性 | panel_template.ts |
| 伤害计算 | 计算技能伤害 DPS | damage_template.ts |
| 异常系统 | 腐蚀/导电/碎甲等 | calculator.ts |
| CLI 工具 | 命令行交互 | panel_cli.ts |

## 关键特性

1. **能力值系统** - 主/副属性提供攻击力加成
2. **乘区分离** - 伤害计算分区明确
3. **珠子系统** - 武器技能等级加成
4. **异常机制** - 多种异常状态影响伤害
