# 伤害计算体系

## 伤害公式

```
最终伤害 = 攻击力 × 技能倍率
  × (1 + 增伤区)
  × (1 + 易伤区)
  × (1 + 增幅区)
  × (1 + 连击区)
  × (1 + 失衡易伤区)
  × 暴击期望倍率
  × 续航倍率
  ÷ (1 + 防御减免)
  × (1 - 抗性减免)
```

## 乘区划分

### 角色侧乘区

| 乘区 | 默认值 | 来源 |
|------|--------|------|
| 基础伤害区 | ATK × 倍率 | 面板攻击力 |
| 暴击区 | 1 / (1+暴伤) / 期望值 | 暴击率/暴击伤害 |
| 伤害加成区 | 1 + Σ加成 | 元素/技能类型/全伤 |
| 伤害减免区 | 1 | 预留 |
| 增幅区 | 1 | 预留 |
| 连击增伤区 | 1 | 预留 |
| 特殊乘区 | 1 | 预留 |

### 敌方侧乘区

| 乘区 | 默认值 | 说明 |
|------|--------|------|
| 脆弱区 | 1 | 目标受到伤害提高 |
| 易伤区 | 1 | 预留 |
| 防御区 | 0.5 | 当前固定口径 |
| 失衡易伤区 | 1 | 失衡状态增伤 |
| 抗性区 | 1 | 抗性减免 |

## 伤害加成区规则

### 加成分类

1. **无条件加成** - 默认全部计入
2. **有条件加成** - 需满足触发条件
3. **他人加成** - 来自其他角色的辅助

### 命中判断原则

```
是否命中 = 加成可作用性 × 触发状态
```

**判定规则：**
- 寒冷伤害加成 → 寒冷伤害命中
- 战技伤害加成 → 战技伤害命中
- 必须同时满足"命中"和"已触发"

---

## 多段伤害计算

```text
总伤(不暴击) = Σ 非暴击段伤_i
总伤(暴击)   = Σ 暴击段伤_i
总伤(期望)   = Σ 期望段伤_i
```

---

## 异常伤害计算

```
异常伤害 = 基础伤害区 × 暴击区 × 伤害加成区 × 伤害减免区
         × 易伤区 × 增幅区 × 虚弱区 × 庇护区 × 脆弱区
         × 防御区 × 失衡易伤区 × 抗性区 × 特殊乘区
         × 等级系数区 × 源石技艺强度区
```

### 等级系数

- 其他伤害: 1
- 法术异常/爆发: `1 + (等级-1) / 196`
- 物理异常: `1 + (等级-1) / 392`

### 源石技艺强度

```
源石技艺强度系数 = 1 + 源石技艺强度 / 100
```

---

## 抗性计算

```
总抗性 = 基础抗性 + 元素抗性 - 抗性穿透 - 腐蚀减抗
抗性减免 = 总抗性 / (100 + 总抗性)  (正抗性)
抗性减免 = 总抗性 / 100              (负抗性)
```

### 敌人抗性等级

| 等级 | 抗性值 |
|------|--------|
| D | 0% |
| C | 20% |
| B | 50% |

## 常用默认值

- 技能等级: 9（专3才用M3）
- 百分比: 小数表示 (53.8% = 0.538)
- 防御减免: 0.5 (50%)
