# UI 修改 0.2.0

## 修改日期
2026-04-07

---

## 1. 设计调性

**专业游戏 UI 风格 —— 简洁赛博朋克 + 轻量化数据可视化**

采用低饱和度中性色调搭配高对比度边框，营造专业级战斗模拟器视觉体验。设计语言强调功能性与可读性，通过精细的线条与点状元素构建音乐谱面般的节奏感。

## 2. 色彩系统调整

### 背景色
| 项目 | 色值 | 说明 |
|------|------|------|
| 画布背景 | `#F0F0F0` | 浅灰，柔和不刺眼，突显内容层 |

### 线条与节点
| 项目 | 色值 | 说明 |
|------|------|------|
| 谱线颜色 | `#E5E5E5` | 极浅灰，与背景层次分明但不喧宾夺主 |
| 伤害节点颜色 | `#E5E5E5` | 与谱线同色，保持视觉统一 |
| 谱线粗细 | `2px` | 细线风格，现代感 |
| 伤害节点直径 | `2px` | 微点风格，极简 |

### 边框
| 项目 | 色值 | 说明 |
|------|------|------|
| 画布边框 | `#1A1A1A` | 深黑，近乎纯黑，高对比度锚定视觉边界 |

## 3. UI 修改对照表

| 元素 | 原设计 | 新设计 | 说明 |
|------|--------|--------|------|
| 画布背景 | `#ffffff` 或场景色 | `#F0F0F0` | 浅灰背景降低眼部疲劳 |
| 谱线颜色 | `#000000` 黑色 | `#E5E5E5` 浅灰 | 降低视觉权重，突出技能节点 |
| 谱线粗细 | `4px` | `2px` | 细线更精致 |
| 伤害节点颜色 | 元素色或黑色 | `#E5E5E5` 浅灰 | 与谱线统一，技能按钮提供颜色焦点 |
| 伤害节点直径 | `8px` | `2px` | 微点，极简风格 |
| 画布边框 | `#000000` | `#1A1A1A` | 深黑略柔和于纯黑 |

## 4. 视觉层次说明

```
视觉权重: 高 ──────────────────────────────── 低

技能按钮 (彩色高亮) > 画布边框 (#1A1A1A) > 谱线 (#E5E5E5) > 背景 (#F0F0F0) > 节点 (#E5E5E5)

焦点: 技能按钮承载所有色彩信息
背景: 无彩色系，纯功能划分
边框: 锚定画布边界，重量感
```

---

## 5. 设计原则

1. **功能驱动色彩** - 技能按钮是唯一色彩载体，承载技能类型与元素信息
2. **线条作为结构** - 谱线构建音乐谱面节奏感，细而轻
3. **点作为节拍** - 伤害节点微小精确，强化时间轴定位感
4. **留白创造呼吸** - 低饱和背景给予各元素充分视觉空间
