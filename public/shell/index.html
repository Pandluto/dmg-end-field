当前工作流
Web 负责业务编辑，Shell 负责本地资源。应用存档后刷新 Web 主界面即可生效。
1. 在浏览器 Web 主界面编辑数据，Shell 会同步为当前数据。
2. 在 Shell 数据页把当前数据保存为本机存档或共享存档。
3. 在 Shell 选择存档并应用，刷新 Web 后生效。
