当前工作流
Web 负责业务编辑与排轴恢复，Shell 负责本地资源与旧档迁移诊断。
1. 在浏览器 Web 主界面编辑数据，工作副本会写入 user.sqlite。
2. 在 Web 主界面创建、选择或恢复排轴文档与恢复点。
3. 如有旧本机/共享文件，在 Shell 数据页查看迁移结果并按需重试。
