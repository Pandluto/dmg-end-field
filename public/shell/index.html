当前工作流
Web 负责业务编辑，Shell 负责本地资源。导入数据后刷新 Web 主界面即可应用。
1. 在浏览器 Web 主界面编辑数据，Web 会写入 now-storage 快照。
2. 在 Shell 数据页导出快照，保存为 localdata 或 sharedata 存档。
3. 在 Shell 选择存档写回 now-storage，刷新 Web 后生效。
