## 基质（珠子）

基质（俗称珠子）是武器的附加系统，提供额外的技能等级加成。

**珠子编码规则：**
- 三位数字分别对应三个武器的技能等级
- 例如：663 表示 敏捷提升·大 6级 / 暴击率提升·大 6级 / 夜幕·余辉未消 3级
- 珠子等级上限：663（三个技能全满）
- 珠子起始等级：111（三个技能全为1级）
- 可无基质

**武器技能最终等级计算：**
```
武器技能等级 = 武器技能基础等级 + 珠子技能等级
```
但最终等级不能超过当前阶段的上限等级。
