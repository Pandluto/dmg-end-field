## 四、阵容 & 配队俗语

| 玩家说法 | 含义 |
|----------|------|
| 双先锋 | 两名先锋（职业）角色组成的阵容 |
| 火队 | 以火属性伤害为核心输出的队伍 |
| 轮椅 | 操作简单、容错率高的阵容/打法 |
| 排轴 | 技能释放顺序的时间轴安排 |
| 循环 | 技能循环，指打完一轮后能无缝进入下一轮 |
| 无限火力轴 | 能够无限循环的充能-爆发交替打法 |
| 简易轴 | 稳定性高、操作简单的打法 |
| 进阶轴 | 操作难度高但启动更快的打法 |
