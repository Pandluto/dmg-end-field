## 一、阵容概述

大家好，我是月咒。本期带来莱万汀火队排轴教学。

**阵容**：莱万汀（42）+ 狼卫 + 艾尔黛拉（小羊）+ 安塔尔

核心思路非常简单——本质上是一套**三保一阵容**：
- 狼卫负责给敌人挂火
- 艾尔黛拉和安塔尔提供各种增益（拐力）
- 大家一起给莱万汀打工，让莱万汀负责爆炸输出

### 热启动 vs 冷启动

- **热启动**：操作比较无脑舒服——主 C 开大，辅助给增益，按住左键输出，连携亮了就按。莱万汀吃四层熔火后打强化战技爆发，整体流程非常丝滑，属于轮椅打法。
- **冷启动**：稍有要求。莱万汀释放强化战技时如果 boss 突然位移，需要用**闪避取消技能**——一旦打空就损失四层熔火充能。另外释放连携技时也要尽可能取消后摇，缩短启动时间。

### 本期装备思路

莱万汀携带动火用 + 生物辅助面板，不管零潜还是满潜都可以用。通过更精细、规律的排轴方式，减少安塔尔身上的一件充能装，进一步提高伤害。简单理解就是：**少带充能，多换伤害**。

## 二、热启动·全满大打法

以白垩界卫为例。

1. **安塔尔**的战技给 boss 挂 60 秒**脆弱**。
2. 主控打重击，触发**艾尔黛拉的连携技**，挂**腐蚀**。
3. **全员开大**：莱万汀 → 狼卫 → 安塔尔。艾尔黛拉的大招留着回血，或打断红圈/打失衡。
4. **莱万汀**连携技并闪避取消后摇。
5. **艾尔黛拉**的战技把腐蚀转化为 30 秒脆弱。

### 开大后循环

- 莱万汀开大普攻第三段时给 boss 挂灼热附着 → 触发狼卫和艾尔黛拉的连携技（挂火）。
- 莱万汀普攻最后一段（重击）把 boss 身上的灼热附着转化为自身熔火。
- 血条左侧四个花瓣图标，**四个全红 = 四层熔火满**。
- 打莱万汀**强化战技**输出，按住普攻不松手。
- 连携技亮了就按。

> **大招快结束时**：
> - boss 残血 → 直接用强化战技斩杀
> - boss 血还多 → **留四层熔火**，等大招结束后再用强化战技充能（莱万汀充能主要靠强化战技和连携技）

## 三、冷启动教学

### 充能阈值

| 潜能 | 充能方式 |
|------|---------|
| 零潜 | 2 次强化战技 + 3 次连携技 充满 |
| 满潜 / 零潜穿充能衣 | 2 次强化战技 + 2 次连携技 充满 |

> 两种打法一样，只有细微区别。不管几潜，学这个冷启动就对了。

---

### 第一波充能轴

1. **安塔尔**战技挂 60 秒脆弱。
2. 主控**莱万汀**打重击，触发**艾尔黛拉**连携技。
3. **狼卫**战技挂 1 层灼热附着 → 同时触发狼卫和安塔尔的连携技 → boss 身上共**3 层灼热附着**。
4. 莱万汀普攻最后一段（重击）吸收 3 层火 → **3 层熔火**（图标三瓣白色）。
5. **莱万汀**连携技（闪避取消后摇）→ +1 层熔火 → **四层熔火**。
6. 莱万汀**强化战技**进行充能。

> 打重击恢复技力，等待三人连携技冷却转好。

---

### 第二波充能轴

启动时机：左下角队伍栏蓝色条上的白条（代表连携技冷却），等三人冷却接近转好时启动。

1. 主控**莱万汀**打重击，触发**艾尔黛拉**连携技并使用。
2. **狼卫**战技 → 同时触发狼卫和安塔尔连携技 → boss 3 层灼热附着。
3. 艾尔黛拉连携腐蚀触发莱万汀连携技，直接使用 → +1 层熔火。
4. 莱万汀重击吸收 3 层灼热附着 → **四层熔火**。
5. 在艾尔黛拉腐蚀消失前，用艾尔黛拉战技把腐蚀转化为脆弱。

**此时注意**：

| 配置 | 下一步 |
|------|--------|
| 零潜 + 一件充能衣 / 满潜 | 直接打强化战技 → 大招充能完毕 |
| 零潜 + 推荐输出装 | 等莱万汀连携技冷却好 → 打强化战技 → 触发连携技 → 完成充能（共 2 次强化 + 3 次连携） |

---

### 全员开大爆发

四人大招全部充能完毕 → 莱万汀开大 → 狼卫开大 → 艾尔黛拉开大 → 安塔尔开大。

然后无脑输出：按住左键，连携亮了就按，四层熔火就打强化战技。莱万汀连携技注意取消后摇。

> **大招快结束时**：手里有四层熔火 → 留！不要打强化战技，等大招结束后用强化战技充能。

## 四、进阶技巧

### 技巧 1：补充能

莱万汀大招充能一般是 2 次强化战技 + 2~3 次连携技。实战不可能每把轴都完美——差一丝充能时：

- 艾尔黛拉连携技挂腐蚀 → 触发莱万汀连携技
- 狼卫大招挂燃烧 → 触发莱万汀连携技
- 莱万汀强化战技 → 触发自己连携技

### 技巧 2：连携技取消后摇

每一次莱万汀打连携技都要用**闪避取消后摇**，非常重要。

### 技巧 3：取消强化战技

打强化战技时 boss 位移 → 立刻按闪避取消！虽然消耗技力，但保留四层熔火，节省重新挂火时间。

### 技巧 4：处决切人

boss 失衡时莱万汀开大无法处决 → 切狼卫或其他队员打处决。同时莱万汀有四层熔火时可以在这时打强化战技。处决完再切回来。

### 技巧 5：狼卫三火连招（满潜）

狼卫满潜有大招时，一个人就能挂三层火：

```
狼卫战技 → 狼卫连携技（后空翻）→ 落地前开狼卫大招 → 再次触发连携
```

一次挂三层灼热附着。

## 五、装备养成推荐

### 5.1 莱万汀（42）

| 方案 | 配置 | 说明 |
|------|------|------|
| 极限伤害套 | 零潜打 3 连携、满潜打 2 连携充满大招 | 只为伤害拉满，非常推荐 |
| 新手充能套 | 3 件动火用 + 充能衣服 | 两件衣服二选一，攻击高一点 / 血量高一点，区别不大 |

---

### 5.2 狼卫

- 主要就是点火，技能点可以不用加。
- 满足充能阈值情况下，装备自由搭配。
- 推荐**四充能装**。穿 3 件长息是因为套装 +1000 血 + 力量（攻击/血量），穿散件充能也可以。
- 核心：**狼卫大招越快越好**。

---

### 5.3 艾尔黛拉（小羊）

- 3 件长息 + 生物辅助胸甲。
- 既能保证充能，也能保证奶量。
- 技能只加连携技即可。

---

### 5.4 安塔尔

- **装备**：3 件拓荒 + 纾难印章，满足充能效率并减少连携技冷却。
- 安塔尔的连携技冷却 25 秒——为了和狼卫连携打联动，需要减连携冷却的套装属性。
- **技能**：只加战技和大招。
- **充能**：越高越好，有足够多精炼珠子可以考虑精炼充能效率。
