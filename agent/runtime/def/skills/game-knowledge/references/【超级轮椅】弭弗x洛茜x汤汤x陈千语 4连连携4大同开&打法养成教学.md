# 【超级轮椅】弭弗x洛茜x汤汤x陈千语 4连连携4大同开&打法养成教学

>
> 阵容：弭弗（主C） / 洛茜 / 汤汤 / 陈千语
>
> 类型：超级轮椅阵容，高颜值高强度，四连携四大同开，放空大脑懒人队。

---

## 一、阵容概述

大家好，我是月咒，专注终末地战斗攻略的主播。

今天带来一个**超级轮椅**的阵容——不仅强度在线，颜值也非常大。这套阵容最夸张的地方在于：**只需要按两下战技就能打出四连携、四个大招同时开启**。玩起来不仅放空大脑，而且感觉自己操作贼溜。

可以说这是目前终末地里少有的高强度懒人队：操作简单，爆发直接。

---

## 二、热启动排轴教学

> 热启动用满潜号演示，冷启动用 0+1 号演示。0+1 弭弗实测大概一分钟左右干掉 boss，伤害也非常高。不管什么潜能打法都一样，不用担心。
>
> 这个队伍谁主控都可以，但因为没有治疗，建议用弭弗。

### 2.1 第一波爆发

1. **弭弗**一段战技突进 → 二段战技，给 boss 挂**1 层破防**。
2. **汤汤**的战技给 boss 挂**1 层寒冷附着**。
3. 同时触发三个人的连携技——**陈千语、汤汤、洛茜**同时使用。
4. **洛茜**的二段连携技，此时 boss 身上已有**四层破防**。
5. **弭弗**的连携技给 boss 挂物理脆弱，同时解锁弭弗二段战技。
6. **弭弗**二段战技**猛击**直接爆掉四层破防。
7. **弭弗**三段战技追加伤害，完成这波爆发。

---

### 2.2 第二波：四大连开

第一波打完继续第二波，直接**四大连开**——大招顺序不影响，怎么开都行。

1. **弭弗**开大，给 boss 挂 1 层破防。
2. **汤汤**开大定 boss。
3. **洛茜**开大打输出，顺便挂上灼热附着。
4. **陈千语**开大给伤害和失衡。
5. 触发并使用**陈千语的连携技**，给 boss 挂 1 层破防。
6. **洛茜**连携一段 → 连携二段，给 boss 挂 2 层破防 → boss 四层破防。
7. **弭弗**连携技挂物理脆弱。
8. **弭弗**二段战技 → 三段战技完成斩杀。

---

## 三、冷启动排轴教学

冷启动和热启动打法基本一样。冷启动要打**三轮一模一样的连招**大招才能充好——不过这个轴在充能阶段伤害就非常高了，不是很依赖大招。只需要记住其中一段就学会了。

> 这组号是 0+1，大家可以感受一下伤害。

### 第一轮

1. 弭弗一段战技突进 → 二段战技猛击，挂 1 层破防。
2. 汤汤战技挂 1 层寒冷附着。
3. 同时触发陈千语 + 汤汤 + 洛茜的连携技，全部按掉。
4. 洛茜二段连携技 → boss 四层破防。
5. 弭弗连携技 → 弭弗二段战技猛击爆掉破防 → 三段战技结束。

> 打重击躲技能、恢复技力。下一轮需要**180 技力**（接近两格）才能启动。

### 第二轮（同上）

1. 弭弗一段战技突进 → 二段战技猛击，挂 1 层破防。
2. 汤汤战技挂 1 层寒冷附着。
3. 陈千语 + 汤汤 + 洛茜的连携技，全部按掉。
4. 洛茜二段连携技挂第四层破防。
5. 弭弗连携技挂物理脆弱 → 二段战技猛击 → 三段战技追加伤害。

### 第三轮（同上）

一模一样连招：弭弗战技一段突进二段猛击 → 挂破防 → 汤汤战技 → 三人连携 → 洛茜二段连携 → 弭弗连携 → 弭弗二段战技 → 三段战技。

然后**全员开大**。当充好的时候怪已经死了——如果还有血，四大连开就接到热启动教学的步骤继续循环。

## 四、替补队友

这个阵容尽可能不要变。

- **卡缪**出来可以替换汤汤的位置，卡缪可以拐到 3 个人，队伍战斗力会大幅度提升。
- 没有汤汤的话，暂时用**秋栗**替补。
- **陈千语**可以换成**余烬**或**骏卫**。
- 剩余两个人（弭弗、洛茜）**不能变**——变了就是另外的体系了。

> 换替补后，打法和之前教学一模一样。

## 五、装备养成推荐

### 5.1 弭弗

- **装备**：3 件旧锋 + 点剑定位信标，可以放心精炼。
- **技能加点**：主加战技。
- **武器优先级**：专武 > 典范 > 幻想苦痛。

---

### 5.2 洛茜

| 方案 | 配置 | 特点 |
|------|------|------|
| 极限暴击套 | 两个暴击部件，暴击拉满 | 稳定暴击输出 |
| 极限伤害套 | 一个暴击部件换成终结技增伤，降低暴击率 | 解锁大招伤害上限 |

两套选择哪套都可以。

---

### 5.3 陈千语

- 推荐**4 点剑**，比较通用，可以随时调整装备。
- 这个阵容里陈千语**不需要充能衣服**，大招就可以循环。
- 护手想换的话可以换成散件护手——敏捷 + 终结技伤害的护手。

---

### 5.4 汤汤

| 方案 | 适用条件 | 配置 |
|------|---------|------|
| 充能套 | 满潜以下 | 需要一点充能阈值，穿充能衣服 |
| 全输出 | 满潜 | 直接穿全输出 |
| 纯辅助 | — | 3 件长息给队友拐力，频繁大招挂附着，每轮节省 80~100 技力，充当先锋使用 |

> 汤汤在这个阵容里输出占比约 10%，穿长息纯辅助也很实用。
