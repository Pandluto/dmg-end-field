## 一、秋栗 vs 安塔尔

上个视频讲的是火队带安塔尔的打法，这期补一个带秋栗的攻略。

| | 安塔尔 | 秋栗 |
|------|------|------|
| 偏向 | 对单 | **对群** |
| 战技 | 60 秒脆弱，只能挂一个目标 | 小范围群体火附着 |
| 启动 | 常规 | **更快**，挂火更舒服 |
| 大招 | — | 点天赋后触发**连击效果** |

### 秋栗连击效果

秋栗开大后，下一个战技或大招获得额外伤害加成：

| 下个技能 | 伤害提升 |
|----------|---------|
| 战技 | +30% |
| 大招 | +20% |

> 这个伤害加成是**纯净乘区**（独立乘区）。秋栗开大后，莱万汀接强化战技可以直接额外 +30% 伤害，爆发非常香。

在对群环境下，秋栗战技 + 狼卫连携（也是群体火附着）可以很快给小怪群同时挂上两层灼热附着 → 莱万汀更高频率触发强化战技 → 更轻松叠四层熔火 → 更快开出大招。

## 二、冷启动教学

大招充能是两套一模一样的连招，记住一套就行。

### 第一波充能轴

1. **秋栗**的战技 + **狼卫**的战技 → boss 挂**两层灼热附着**。
2. 触发狼卫连携技 → 挂**第三层火**。
3. **莱万汀**打重击吸收三层灼热附着 → **三层熔火**（花瓣三个白）。
4. 艾尔黛拉连携延迟爆炸后 → 触发莱万汀连携技 → 使用后闪避取消后摇 → +1 层 → **四层熔火**。
5. 满熔火 → 莱万汀**强化战技**消耗熔火，增加伤害和恢复大招能量。

---

### 第二波充能轴

> 启动前看左下角狼卫连携冷却（蓝条上的白条满了 = 好了）。如果打狼卫三火连招可节省秋栗 100 技力，但刚需狼卫五潜 + 手速。

本轮需要加入艾尔黛拉战技挂脆弱，其他和第一波一样。

1. **秋栗**战技 + **狼卫**战技 → 两层灼热附着。
2. 狼卫连携技 → 第三层火。
3. 莱万汀重击吸收三层火 → 三层熔火。
4. 触发艾尔黛拉连携技 → 触发莱万汀连携技 → +1 层熔火 → 四层熔火。
5. **艾尔黛拉**战技把腐蚀转为脆弱。

**此时分情况**：

| 配置 | 下一步 |
|------|--------|
| 零潜 + 充能衣 / 满潜 | 1 次强化战技 → 大招充能完毕（2 强化 + 2 连携） |
| 零潜 + 生物辅助面板（无充能） | 等莱万汀连携技转好 → 打强化战技 → 共 2 强化 + 3 连携充满 |

---

### 全员开大爆发

boss 身上有艾尔黛拉脆弱 + 专武异常 → 技力充足时，莱万汀强化战技 + 连携技充满大招。

1. **全员开大**：莱万汀 → 狼卫（打失衡） → 艾尔黛拉（打失衡）。
2. 按住左键不松手，连携亮了就按。
3. 四层熔火时 → **秋栗开大**触发连击效果。
4. 莱万汀**强化战技**（吃连击 +30% 伤害） → 莱万汀连携技。
5. 继续循环。斩杀判断：
   - 能斩杀 → 大招期间打强化战技
   - 不能 → 留四层熔火，等大招结束后打强化战技充能

## 三、热启动

热启动只需要打一个爆发前置，就可以无脑连招。

### 前置

1. 主控打重击 → 触发**艾尔黛拉**连携技，挂腐蚀。
2. **狼卫三火连招**：战技 → 连携（后空翻） → 落地前开大招打断 → 大招结束后再次触发连携。

> 需要狼卫五潜。没满潜也可以打，就是少一层火。

3. **艾尔黛拉**战技把腐蚀转脆弱。
4. 打重击把火吸到手里。

### 爆发

1. 莱万汀直接开大（手里四层熔火）。
2. **秋栗**开大触发连击效果。
3. 莱万汀强化战技（吃连击 +30%）。
4. 按住左键 → 连携好了就按 → 四层熔火打强化战技 → 循环。

## 四、进阶技巧

### 技巧 1：补充能

大招差一丝充能时：艾尔黛拉连携挂腐蚀 / 狼卫大招挂燃烧 / 莱万汀强化战技 → 均可触发莱万汀连携技。

### 技巧 2：连携技取消后摇

每次莱万汀打连携技都用闪避取消后摇。

### 技巧 3：取消强化战技

boss 位移时立刻闪避取消——消耗技力但保留四层熔火。

### 技巧 4：处决切人

boss 失衡 → 切狼卫处决。莱万汀有四层熔火时可同时打强化战技。

### 技巧 5：狼卫三火连招

满潜有大招时：战技 → 连携（后空翻） → 落地前大招 → 再次触发连携。一次三层火。

### 技巧 6：大招期间连携技增加时间

莱万汀开大期间使用连携技，可以**少量增加大招持续时间**。

### 技巧 7：普攻/强化战技暂停大招计时

莱万汀普攻或强化战技期间，**大招时间暂停**，不占用大招持续时间。

## 五、装备养成推荐

### 5.1 莱万汀（42）

| 方案 | 配置 | 说明 |
|------|------|------|
| 极限伤害套 | 零潜 3 连携、满潜 2 连携充满 | 只为伤害拉满，非常推荐 |
| 新手充能套 | 3 件动火用 + 充能衣服 | 更简单，推荐新手 |

---

### 5.2 狼卫

- 主要挂火，技能点可以不加（萌新省资源）。
- 不打三火连招时，此阵容需要用两次秋栗战技挂火，比安塔尔队多消耗 100 技力启动。萌新推荐**天灾防护套**（额外 +50 技力，缓解启动压力）。
- 第二套：3 件长息（+1000 生命值，不触发增伤效果，单纯吃生命加成）。
- 经常打三火连招 → 充能阈值堆到 **90%+**（面板显示 195%+）。狼卫充能越高越好。

---

### 5.3 艾尔黛拉（小羊）

- 3 件长息 + 生物辅助胸甲，保证充能和奶量。
- 技能只加战技即可。

---

### 5.4 秋栗

- **装备**：**四充能装**。秋栗连击触发需要敌人进入失衡节点，实战不容易稳定对轴 → 四充能提升大招充能效率，循环更稳定。
- **技能**：只需要加大招，点到 9 级基本够用。装备也不需要精炼，萌新放心养成。
