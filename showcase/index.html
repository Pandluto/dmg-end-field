  1. 01攻略检索def_data_game_knowledge 找到碎冰队资料，并锁定“装备养成推荐”章节。
  2. 02章节精读def_data_game_knowledge_section 只读取指定章节，保留推荐依据和边界。

  5. 05追问与复核继续核对弭弗武器、阿列什 3+1 装备与充能数值；每个结论都能回到节点与会话。

LIVE PRODUCT SURFACES

不是功能名列表。
这是实际打开后的工作面。

以下全部来自本地运行的工作台：四人选队、已排技能轴、可计算报表、资料工作表、资源库、恢复历史与工作节点树。每个区域都有自己的操作、数据与导入导出边界。

01 / TEAM → TIMELINE

队伍进入排轴后，
技能不再是一张静态表。

最多四名干员构成出战队列。排轴画布按分组组织 A / B / E / Q 技能节点；每个节点都能继续承载 Buff、状态、敌方抗性、命中微调和伤害详情。

四人出战队列技能节点拖拽与分组批量 Buff 操作目标抗性 / Hit 微调

01.5 / TIMELINE → BUFF WORKBENCH

Buff 既能批量施加，
也能逐段追溯。

排轴上的节点并非只负责排布顺序：可按干员框选并批量增删 Buff，也可打开单个技能，查看生效条件、期望伤害、暴击 / 非暴击结果，以及每一个伤害乘区的计算依据。

框选与干员筛选29 个技能节点Buff 增删与替换逐段伤害乘区

02 / CALCULATE → REPORT

计算之后，
还要把过程讲清楚。

LOCAL DATA STUDIO

资料不是只读素材；它们本身也是可维护的工作表。
